.NET ASPIRE 预览版 7 发布
====================

.NET Aspire 预览版 7 并不是原计划的一部分，此预览版有很多重大 API 更改，部分原因是一旦产品发布，我们将致力于稳定的 API 表面。可以说，Aspire团队希望确保在最终发布之前完成这些 API 更改。 但作为开发人员，我们会喜欢能够对快速发展的开发生态系统中的变化做出快速反应。

.NET Aspire 预览版 7 的发布版本为 8.0.0-preview.7.24251.11。这个版本包含了大量的 API 破坏性更改，主要是为了在最终发布前确保 API 的稳定性。这些更改包括属性重命名、类重命名和方法签名更改。例如，组件设置类中的共享属性名称进行了调整，以增加一致性和易用性。此外，Azure Service Bus 组件的某些属性名称也进行了更新，以更好地反映其代表的含义。

此外，.NET Aspire 的仪表板在这个预览版中也得到了更新，改善了视觉效果并增加了更多本地化内容。容器镜像的默认设置也有所更改，现在所有容器镜像都完全限定了其注册中心和标签。

.NET Aspire 预览版 7 的 GitHub 发布页面提供了关于这个版本的详细信息。这个版本包含了许多更新和改进，例如对 Azure.Identity 的更新、对 ASP.NET Core 的升级、对 OpenTelemetry HTTP 监测包的更新、对组件 API 的更改、对 Docker 镜像标签的更新，以及对各种组件和服务的修复和改进。此外，还进行了许多其他的修复和改进，包括对文档的更新和对测试的增强。这些更改旨在提高性能、稳定性和用户体验。有关完整的更改列表和详细信息，请查看 [GitHub 发布页面](https://github.com/dotnet/aspire/releases/tag/v8.0.0-preview.7.24251.11)。
