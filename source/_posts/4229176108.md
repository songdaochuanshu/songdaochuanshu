TIOBE 11月榜单：Java和 C# 之间的差距缩小到0.7
================================

TIOBE 公布了 2023 年 11 月的[编程语言排行榜](https://www.tiobe.com/tiobe-index/)。 虽然这期重点介绍的是Kotlin，本月，它的排名上升了 0.17%，从第 18 位上升到第 15 位，前进了 3 位。 [TIOBE的10月份榜单](https://www.oschina.net/news/260951/tiobe-index-2023010)，C# 和 Java 之间的差距越来越小，达到了历史新低，10月份两者的差距仅为 1.2%。 这一趋势在11月继续保持， 目前两者差距缩小到了0.7%。

C# 相比Java，最大的优势在于，可以非常高效的用在工业界，对接大量已存在的C, C++代码, 但是Java就不行。

Java的基本盘：互联网，现在已经慢慢在走下坡路了。所以，Java的岗位已经没多少增量了。 而C# 就是更好的C++，它不但可以用在互联网服务器上，还可以写游戏客户端，写工业软件，写机器控制UI... 在中国产业生态位上升的今天，C# 的增量比Java的多。

C# 十分接近于传统高级语言 C 和 C++，是一门面向对象的编程语言，但是它与 Java 非常相似，有许多强大的编程功能，因此得到广大程序员的青睐.
