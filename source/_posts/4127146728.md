1\. 热门开源项目
----------

### 1.1 可自托管的网页存档平台：ArchiveBox

![](https://img2024.cnblogs.com/blog/759200/202410/759200-20241021223150494-947717108.png)

**主语言：Python**，**Star：21k**，**周增长：500**

这是一款基于 Python 的网页存档工具，专为帮助用户保存和归档互联网内容。它免费、开源且可自托管，支持从 URL、浏览器书签、RSS、开源项目和社交媒体等资源下载网页，生成本地快照以便离线阅读和归档，并提供 HTML、PDF、图片和文本等多种存档格式。著名的互联网档案馆（Wayback Machine）就基于该项目构建而成。

> GitHub 地址→[github.com/ArchiveBox/ArchiveBox](https://github.com/ArchiveBox/ArchiveBox)

### 1.2 灵活易用的 TLS 库：mbedtls

![](https://img2024.cnblogs.com/blog/759200/202410/759200-20241021223154517-1136267424.png)

**主语言：C**，**Star：5.4k**，**周增长：150**

该项目是用 C 语言编写的加密库，可用于实现加密原语、X.509 证书操作以及 SSL/TLS 和 DTLS 协议。它体积小巧、易于移植，并提供丰富的示例，适用于物联网和嵌入式系统等资源有限的设备。

> GitHub 地址→[github.com/Mbed-TLS/mbedtls](https://github.com/Mbed-TLS/mbedtls)

### 1.3 macOS 的虚拟显示器：DeskPad

![](https://img2024.cnblogs.com/blog/759200/202410/759200-20241021223158836-1248066688.jpg)

**主语言：Swift**，**Star：4.9k**，**周增长：3k**

这是一款适用于 macOS 的屏幕共享虚拟显示器。它可以创建一个虚拟显示器，将屏幕内容以窗口形式镜像显示，支持调整屏幕分辨率，非常适合屏幕共享场景。

> GitHub 地址→[github.com/Stengo/DeskPad](https://github.com/Stengo/DeskPad)

### 1.4 开源的电子邮件营销平台：listmonk

![](https://img2024.cnblogs.com/blog/759200/202410/759200-20241021223202328-623455253.png)

**主语言：Go**，**Star：14k**，**周增长：200**

该项目是采用 Go、Vue.js 和 PostgreSQL 构建的电子邮件营销平台。它功能丰富且开箱即用，支持独立可执行文件和 Docker 运行方式，提供批量导入邮件列表、动态邮件模板、可视化分析，以及 SQL 查询和条件筛选订阅者等功能。

> GitHub 地址→[github.com/knadh/listmonk](https://github.com/knadh/listmonk)

### 1.5 快速构建机器学习应用的利器：Gradio

![](https://img2024.cnblogs.com/blog/759200/202410/759200-20241021223207491-1187760499.gif)

**主语言：Python**，**Star：33k**，**周增长：700**

这是一个用于快速构建和分享机器学习演示应用的 Python 库，无需掌握 JavaScript、CSS 或 Web 部署知识。只需几行 Python 代码，就能创建带界面的 Web 应用。它支持公开访问、复杂的交互和布局，并提供文本框、图像、按钮等 30 多种内置组件。可在 Jupyter Notebook、Google Colab 等多种环境中运行，适用于教学、项目演示和构建机器学习 WebUI 等场景。

> GitHub 地址→[github.com/gradio-app/gradio](https://github.com/gradio-app/gradio)

### 2.1 支持各种流媒体协议的处理工具：go2rtc

![](https://img2024.cnblogs.com/blog/759200/202410/759200-20241021223211174-1270619663.png)

**主语言：Go**

这是一个用 Go 语言编写的库，支持 RTSP、WebRTC、HomeKit、FFmpeg、RTMP 等视频流协议的处理。在 FFMPEG 的加持下，它几乎能将任何媒体格式作为输入源，转换为适用于主流流媒体服务和浏览器的格式。

> 项目详情→[hellogithub.com/repository/3bdcd4d189d34008a828e89cccc1ab54](https://hellogithub.com/repository/3bdcd4d189d34008a828e89cccc1ab54)

### 2.2 在浏览器中运行 Postgres 数据库：pglite

![](https://img2024.cnblogs.com/blog/759200/202410/759200-20241021223214451-1791932079.png)

**主语言：TypeScript**

该项目将 PostgreSQL 数据库编译成 WebAssembly (WASM)，并打包成一个 TypeScript/JavaScript 客户端库。它压缩后体积不到 3MB，可以在浏览器、Node.js、Bun 和 Deno 环境中运行，无需安装任何额外的依赖。提供灵活的存储选项，支持内存存储、本地持久化或 IndexedDB。

> 项目详情→[hellogithub.com/repository/cf52acd434ee4691bdc96fe12a16fa15](https://hellogithub.com/repository/cf52acd434ee4691bdc96fe12a16fa15)
