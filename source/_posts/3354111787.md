博客园众包平台：嵌入式开发任务悬赏5万，招募开发者接单
===========================

![](https://img2024.cnblogs.com/blog/35695/202405/35695-20240525213947248-60992660.jpg)

博客园众包平台是我们从今年5月开始的商业化突围项目，一边在以企业微信+自己搭建的 GitLab 简单运营众包业务，一边在[融资](https://www.cnblogs.com/cmt/p/18172236)投入平台系统的建设，当前已召集到1000多位合作开发者，多数是有多年开发经验的资深开发者。

万事开头难，虽然召集了这么多合作开发者，却一边面临僧多粥少的尴尬，一边面临有单无人接的难堪，今天发布的这一单就是后者的一个案例。

这个单子的任务是为一位客户的嵌入式硬件开发 SDK 与标准接口层，这个单子很符合我们软件开发任务众包平台的定位，而且报价也不低。一开始能遇到这样的好单子，是一种幸运，但好运成好事没那么容易。虽然目前召集的合作开发者中有30多位与嵌入式开发相关，虽然发布单子后甚至有2位开发者差点接单成功，但最终还是没有达成合作，我们无奈地关闭了这个单子。

在单子关闭4天后的今天，我们有点不甘心，不能就这么放弃，虽然在合作开发者中找不到人接单，但我们可以尝试公开招募，在更多的开发者中努力寻找，不能让客户失望。我们找客户商量看是否可以让我们专门针对这个单子公开招募开发者，幸运的是客户同意了，接下来就看我们能不能扭转局面反败为胜，于是就有了这篇公开招募博文。

这个单子的需求是这样的：

*   任务描述：为客户的自研固件开发通用 GUI SDK 与 GenICam 驱动（支持 GenICam 标准的接口层）
*   任务说明：客户会提供完整的固件底层 api，开发者只需要调用客户提供的 api，如果有缺少的底层 api，由客户负责开发
*   温馨提示：可以使用 C++ 或者 C# 开发，只要客户能通过源码进行跨平台编译
*   工期：2-3月
*   费用：5w

为了便于理解这个任务，我们拿大模型打个比方（只是我们自己的理解，仅供参考），客户开发了自研的大模型，并且实现了大模型 api，现在想找人基于已有大模型 api 开发 SDK 与兼容 OpenAI api 的 api。从这个角度看，难度不是很大，并不需要很熟悉硬件，只需要熟悉客户提供的固件 api。

如果您熟悉这块的开发并有意向接这个单子，欢迎加[企业微信](https://www.cnblogs.com/cmt/articles/17677736.html)与我们联系。
