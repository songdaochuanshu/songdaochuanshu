想要一个漂亮的博客？Simple Memory - 博客园 cnblogs 个性化博客配置
=============================================

前言
==

地址：[https://www.cnblogs.com/FReQuenter5156/p/setblog.html/](https://www.cnblogs.com/FReQuenter5156/p/setblog.html/)

如题，使用的是 Simple Memory 主题。

Github 连接：[https://github.com/BNDong/Cnblogs-Theme-SimpleMemory](https://github.com/BNDong/Cnblogs-Theme-SimpleMemory)。

想个性化自己的博客只需要改代码就行了。不难改，注释很充分（也就是换一些 url 啥的。反正我是这么改的。

搭建教程
====

基本配置
----

博客皮肤先选择 Simple Memory。

提前开通 JS 权限。

其余的随意。注意在 Simple Memory 中有对行号的渲染，会和自带的行号冲突，需要关掉其中一个。去除 Simple Memory 中的行号就在代码里找吧。注释很全的。

直接把这个博客的代码贴上来吧。

这个是已经删除了一部分非必要的，如果想要最全的可以去 Github 上搞下来。
