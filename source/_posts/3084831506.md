前端周刊第二十三期
=========

前端周刊发表每周前端技术相关的大事件、文章教程、一些框架的版本更新、以及代码和工具。每周定期发表，欢迎大家关注、转载。  
如果外链不能访问，关注公众号**「前端每周看」**，里面有解决办法

大事件
---

*   [离职的互联网人，都去哪儿了？](https://mp.weixin.qq.com/s/yH8iD5E8iakpIMLfV5pMCg)——转行，从接受落差开始。
    
*   [用 JavaScript 解释数学符号](https://runjs.app/blog/mathematical-notation-for-javascript-developers-explained)——数学有自己迷人的符号世界，但如果你觉得它有点不透明，这篇文章会有所帮助。它将许多数学符号领域归结为JavaScript 等价物，从而更容易理解正在发生的事情。
    
*   [“JavaScript Hydration 是一种解决方法，而不是解决方案”](https://thenewstack.io/javascript-hydration-is-a-workaround-not-a-solution/)——许多新框架提供的 hydration 是否是不必要的开销？Angular 的发明者 Miško 解释了他为什么这么认为，并分享了 Qwik 框架使用的策略。
    
*   [Plasmo：“It's Like Next.js for Browser Extensions”](https://github.com/PlasmoHQ/plasmo)——一个面向 React 和 TypeScript 的框架，用于构建您自己的浏览器扩展，包括实时重新加载、自动部署到主要浏览器扩展市场等。

--

1.  [半色调打印的 JavaScript](http://anderoonies.github.io/projects/halftone/) — 一个令人愉快、实用且交互式的演练，介绍了使用基本 JavaScript 和画布再现经典半色调样式打印过程所涉及的内容。
    
2.  [构建框架可互操作的 Web 组件](https://css-tricks.com/building-interoperable-web-components-react/) — 如果您使用不同的框架，如 React、Svelte、Vue 等你要？Web Components 提供了一种方法。
    
3.  [使用 JavaScript 创建超声波支付系统](https://charliegerard.dev/blog/ultrasonic-payments/) - 一个有趣的设备间超声波数据传输实验。
    
4.  [RedwoodJS 在 100 秒内解释](https://www.youtube.com/watch?v=o5Mwa_TJ3HM) - RedwoodJS是一个全栈 JS 框架，由 GitHub 的一位联合创始人提供支持。
    
5.  [通过 UTM 在 Apple Silicon Mac 上运行 Windows/ARM](https://2ality.com/2022/06/windows-on-macs-via-utm.html) - 您是一个 macOS 用户，想要启动 Windows 以了解 Node.js 在那里的工作方式、工具感觉如何等？

代码和工具
-----

1.  [KaTeX 0.16.0：最快的网络数学排版库](https://katex.org/)：数百个表达式要呈现在一个页面上？尽管支持许多 TeX 函数，但也没有问题，也没有依赖关系。
    
2.  [Timeline JS：为 Web 创建有用的时间线](https://timeline.knightlab.com/)：由大学的新闻实验室构建，它的目标对象不是专业开发人员，而是只需要在线讲述故事或展示一些数据的人。
    
3.  [Fastify 4.0 发布：Node.js Webapp 框架](https://medium.com/@fastifyjs/fastify-v4-ga-59f2103b5f0e)：两年来流行的高性能 Node.js Web 框架的第一个主要版本值得庆祝。重点放在了稳定性、现代化和改进已经相当稳定的开发人员体验上，而不是华而不实的新功能，尽管这篇文章确实涵盖了一些更新。
    
4.  [pretty-ms：将毫秒转换为人类可读的字符串](https://github.com/sindresorhus/pretty-ms)： 1337000000→15d 11h 23m 20s。
    
5.  [面向 UI 设计师的最佳 JavaScript 和 CSS 动画库](https://www.sitepoint.com/our-top-9-animation-libraries/)：运行九个动画选项，涵盖优缺点以及何时选择每个选项。
    
6.  [qnm：用于查看node\_modules目录的 CLI 实用程序](https://github.com/ranyitz/qnm)：目标是更轻松地查看node\_modules所选目录中发生的情况，包括让您搜索包。同时支持 npm 和 Yarn。
    
7.  [is-online 10.0：检查 Internet 连接是否正常](https://github.com/sindresorhus/is-online)：适用于 Node.js 和浏览器，并使用不同的方法检查 Internet 是否真的可用。现在是一个纯 ES 模块。
