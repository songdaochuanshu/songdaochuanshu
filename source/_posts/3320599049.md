来啦来啦｜开源 \* 安全 \* 赋能 - .NET Conf China 2022
==========================================

  

[![image](https://img2022.cnblogs.com/blog/510/202211/510-20221102162543296-242861200.png "image")](https://img2022.cnblogs.com/blog/510/202211/510-20221102162542507-440391496.png)

**大会介绍**

.NET Conf China 2022 是面向开发人员的社区峰会，延续 .NET Conf 2022 的活动，庆祝 .NET 7 的发布和回顾过去一年来 .NET 在中国的发展成果，它是由中国各地区的技术社区共同发起举办、知名企业和开源组织联合协办，是年度国内规模最大的 .NET 线上和线下会议，目的是用中文传播 .NET 相关技术的知识和经验。

2014 年微软组织并成立 .NET 基金会，已经有众多知名公司加入 .NET 基金会，在平台项目中，.NET 平台上的 ARM 架构由三星团队贡献，在 .NET Core 3.1 上得到了完整支持，.NET 5 以上得到进一步的优化，龙芯团队已经开发完成 MIPS64 架构和 LoongArch 架构的支持、LoongArch 架构已经合并进入.NET 7，RISC-V 的支持正在开发进行中。

2016 年以来，我们的 .NET 社区通过协作来推动 .NET 开源项目和社区的发展，在 GitHub/Gitee 上活跃着很多中国的 .NET 开源开发者组织，例如 NCC、盛派开发者、龙芯 .NET 等，我们在全国各大城市有 .NET 俱乐部定期举办活动，自疫情以来，更多是线上的直播活动。2019年我们也在上海成功举办了第一届中国 .NET 开发者峰会，2020 年我们在苏州成功举办第二届 .NET 社区的中国峰会，2021年11月20日~ 21日在线上举办第三届 .NET 中国峰会，同时举办了第一届黑客马拉松比赛。

今年，.NET 相关技术社区将一起，于 **2022年12月3日~4日** 举办为期两天的 .NET Conf China，邀请 .NET 中国开发者共襄盛举。今年峰会的主题是--”开源 安全 赋能”。在当前疫情常态化的受限环境中，基于安全与稳定的软件供应链考虑，高效率的 .NET 平台是我们战胜当前复杂环境的有力武器。大会面向企业技术负责人、一线工程师以及社区个人开发者，特别是已在公司或个人项目中实践过 .NET 的开发者以及希望了解 .NET 的所有人员。

**活动形式**

.NET Conf China 2022 是一个社区性质的技术峰会，本次大会以“开源、安全、赋能”为主题，致力于成为中国 .NET 开发者面对面交流的技术盛宴。

本次活动首日以业务，教育为导向，主题演讲将分享最新企业应用趋势，以及 .NET 生态的建设和发展。第2天是由 .NET 开发者和社区带来的议程“. NET ❤ Open Source”，邀请国内外众多重量级大师及专业讲师，分享第一手技术新知识和丰富的实践经验。为中国的 .NET 开发者和企业提供一次充分的成果展示、技术分享、能力提升、行业资讯交流，企业人才储备建设的机会。
