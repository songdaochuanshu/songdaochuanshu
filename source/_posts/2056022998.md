开源项目分享：ChatGPT 控制台聊天应用
----------------------

分享一个我最近完成的一个小应用，一个ChatGPT 的控制台聊天应用，大家都在搞AI，我也来玩一玩，顺便分享到社区，有兴趣的小伙伴可以去我的github主页下载体验。

### 项目简介

**ChatGPT 控制台聊天应用** 基于.NET8实现，一键拥有你的私人 ChatGPT ，支持OpenAI（ChatGPT）；MoonshotAI（Kimi）；DeepSeek等模型。是一个简单易用的命令行界面，它允许用户与 ChatGPT 进行交互，获取信息、解答疑问或仅仅是进行闲聊。每个人都能够轻松上手。

使用这个应用非常简单，只需几个步骤：

1.  **克隆仓库**：通过 Git 克隆项目到本地。
    
        git clone https://github.com/cqloki/CQAI.Chat
        
    
2.  **如何使用**：配置上大模型服务地址，及您在平台上创建的 API Key
    
3.  **开始聊天**：按照提示与 ChatGPT 进行对话。
