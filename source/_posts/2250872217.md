【译】Visual Studio 17.8 中我最喜欢的特性
==============================

对于 Visual Studio 团队来说，这是忙碌的一周，他们准备了 Ignite 和 .NET Conf，并发布了最新版本的 Visual Studio 2022，即17.8版本。有很多很酷的新功能，AI 增强，性能改进等等的公告。这太难以消化了，所以我想我应该让大家注意到我最喜欢的一些特性。

**保留大小写的查找和替换**
===============

　　在进行替换时，现在可以保留代码中每个匹配的原始大小写。请注意，要获得 Pascal 风格和 Camel 风格，您的替换字符串也必须是 Pascal 风格或 Camel 风格。

　　用 Alt+V 或点击“保留大小写”选项切换替换窗口中的大小写保存。

　　下面是在文件中替换(Ctrl+Shift+H)的效果。

![](https://img2023.cnblogs.com/blog/270073/202312/270073-20231208174130838-287191543.png)

　　这是我一直在使用的一个方便的功能。

**富拉请求体验**
==========

　　这个特性已经开发了一段时间，并且是17.7和17.8预览版的一部分。现在它终于来了，而且很壮观。你不仅可以直接从 Visual Studio 内部创建拉取请求，还可以同时获得 GitHub 和 Azure DevOps 的支持。

![](https://img2023.cnblogs.com/blog/270073/202312/270073-20231208174146456-1111765562.png)

　　您可以在描述中键入 markdown，也可以添加审稿人。这是一种丰富而自然的发送拉取请求的方式，它很好地补充了我的工作流程。

　　这是我的两个选择，我想你可能会像我一样喜欢。试试吧，和往常一样，在下面的评论中告诉我你最喜欢的功能是什么。如果你还没有安装 Visual Studio 17.8，那么打开安装程序，点击更新按钮，或者在下载一个最新版的 Visual Studio。
