SQL Server 2017 各版本之间的差异
========================

SQL Server 2017的亮点
------------------

### 您选择的语言和平台

使用您选择的语言在本地和云中（现在在 Windows、Linux 和 Docker 容器上）构建现代应用程序。

### 行业领先的性能

充分利用任务关键型智能应用程序和数据仓库出类拔萃的可扩展性、性能和可用性。

### 漏洞最少的数据库

利用 NIST 漏洞数据库过去七年内评定为漏洞最少的数据库保护静态和动态数据。注：根据美国国家标准与技术研究院所掌握的数据，与其他主要数据库供应商相比，SQL Server 在过去七年发生的安全漏洞最少。

### 实时智能

以高达 1 百万次预测/秒的实时分析速度获取变革性的业务见解。

可用的 SQL Server 2017 版本
----------------------

### Enterprise 版（企业版）

### "高级" 产品/服务 SQL Server Enterprise 版本提供全面的高端数据中心功能，具有超快的性能、无限制的虚拟化和端到端商业智能-可为关键任务工作负荷提供高服务级别，并使最终用户能够访问数据见解。

### Standard 版（标准版）

SQL Server Standard edition 为部门和小型组织提供了基本的数据管理和商业智能数据库，以运行其应用程序，并支持用于本地和云的常见开发工具，同时提供最少的 IT 资源。

### Express 版（快速版）

Express 版本是入门级的免费数据库，是学习和构建桌面及小型服务器数据驱动应用程序的理想选择。它是独立软件供应商、开发人员和热衷于构建客户端应用程序的人员的最佳选择。如果需要更高级的数据库功能，可以将 SQL Server Express 无缝升级到 SQL Server 的其他更高版本。SQL Server Express LocalDB 是 Express 的轻型版本，它具有其所有可编程性功能，在用户模式下运行，并且具有快速的零配置安装和必备组件的简短列表。

### Developer 版（开发人员版）

SQL Server Developer 版本允许开发人员在 SQL Server 之上构建任意类型的应用程序。它包括 Enterprise 版的所有功能，但有许可限制，只能用作开发和测试系统，而不能用作生产服务器。SQL Server Developer 是构建和测试应用程序的人员的理想选择。

### Web版

对于 web 托管商和 web vap 来说，SQL Server Web 版本是低拥有成本的选项，可为小规模 web 属性提供可伸缩性、经济性和可管理性的功能。使用面向公共网站的安全、经济高效且高度可扩展的数据平台 - 仅对第三方托管服务提供商开放。
