第三回 万文疑谋生思绪 璃月港口见清玉

\=====================================================================

云溪愣了下，在他的认知中，神之眼正常而言不会有黑色的存在，就像在实数的体系内不会有i一样，他摇了摇头。而纪存初的眼中有闪过一次迷茫和失望，很快恢复过来，笑道：“算了，反正也只是个坊间传说而已，对了，小子，有兴趣入我万文集舍么？”

话题跳转的如此之快，让云溪有点猝不及防，万文集舍到确实是个大宗级别的存在，进入其中谋个职位倒也算是光宗耀祖的美事，可最重要的事情在于，万文集舍太过于神秘了，以至于云溪都不知道自己如果进去可以干嘛，难道当个书贩子？还是当个推销员？亦或者辅助编撰丛书？

“小子受宠若惊，但不知纪伯看上了我哪一点呢？”云溪将书收到怀里，抱拳示意。

纪存初回道：“当然是你能使用元素力了，毕竟神之眼人人都有，有用的神之眼可不是什么平常物件儿。当然我不是让你直接进入万文集舍，我也不想招那么多议论。”

云溪心中闪过一丝失落，随之释然，却又听纪存初继续道：“我是想引荐你去古华派，飞云一脉两个月前想招收新弟子，我想插个队，把你塞进去。”

云溪震惊的看着纪存初，相比于万文集舍这种神神秘秘的组织，古华派显然要敞亮了许多，而且这几年古华派也在自己山头开展了旅游业，还让人付费瞻仰《飞云天书》，更是在璃月整个社会层面名声大噪。他们本身的实力强大，再加上口口相传的关于他们祖先的侠义故事，无疑让这个宗门变成了很多少年少女梦中圣地。

纪存初没有继续说下去，脸上忽然有些阴晴不定，看了看云溪，又摩挲着下巴，云溪有些激动又茫然的看着这个性情无常的老头子，悄悄往后退了一步，既怕老头子又改口放弃了自己，又怕老头子莫名发怒把自己给宰了，甚至心中默念着，希望自己的老爹能够救场。

大概沉默了半分钟，纪存初咬咬牙，轻咳了一声：“云先生，进来吧，我没把云小子怎么样。”

门吱呀一声打开个缝儿，云老爹的头探进来，尴尬的笑笑：“我刚让我堂客做好了饭，要不纪老先生在寒舍先吃顿饭先？”

“不了，我老婆子做好饭了”纪存初有些明白云老爹的意思，摆摆手：“我想引荐你家小子去古华派，虽然古华今年收徒期过了，不过我给个引荐信就行，还有，我孙子纪芾到时候也会跟着云小子一块儿过去，他比云小子小一岁，还希望照顾照顾他。”

说完看了云溪一眼，大步走到门前，云老爹很识相的打开了门，笑道：“纪老先生，那下次。。”

纪存初仿佛失去了什么兴致，平淡的回道：“好，下次我和云先生好好聊聊吧，我这次就不叨扰您了。”

云老爹看着纪存初走远，脸上冷汗才流下来，赶忙跑到云溪身边，拽来拽去看看有没有缺胳膊少腿啥的，云溪看着自己老爹紧张的样子，有些好笑：“爹，我没事儿，纪伯没怎么样我，还把书送我了。”

云老爹还是不放心，让云溪拿过书来他看下，可是他看半天也没看出个头绪来，又还给了云溪：“溪儿，纪老爷子说你能使元素力驱动神之眼了？能否给爹看下？”

云溪照样将新手引导的那个示例使用了出来。

"我是我"

看着眼前几个闪着光的字，云老爹又是担心又是激动，看着云溪：“溪儿，今天是你成年的日子，也是改变你命运的日子，我云家有福气，也希望溪儿以后能好好过日子，用自己的元素力做正确的事情啊。”

云溪茫然的看着老爹，思索了一下，点了点头：“爹，我会谨记的，你放心。”

云老爹欣慰的笑了笑，拍了拍云溪的肩头：“好，溪儿你一定要记住今天爹给你说的这话。走吧，你娘烧饭了，比往常丰盛，好好吃一顿，明天我把你送到万文集舍去。”

。。。

今天热闹的地方绝对不只有云溪自己的家里，今天的璃月港也比以往热闹的多，不仅仅有着商船来往，还有很多好事者聚在那里看着热闹。

有个人淹死了，被商船的锚钩住了，这才被捞了上来。

虽然被泡的有些浮肿，但看着面向是个十八九的少女，身上的缎带也显示着这不是一个普通人家的孩子，这不禁让人唏嘘，还不到桃李之年就命丧黄泉了，着实是一件令人惋惜的事情。千岩军将尸首所在地围了起来，正在等待专门验尸的人进行现场的初步推算。

忽然，有人惊奇的喊道：“你们看，这个女孩儿的身子缩水了！”

确实，刚捞上来的时候少女的尸体是一种巨人观的状态，面目狰狞，衣服也被紧绷着，但是现在看上去，虽然少女的脸被盖住了，但是明显缩水了很多，衣服也逐渐坍塌下来，倒是显得瘦弱了很多。千岩军这下子开始着急了起来，专业的人还没到，自己人也无法私自去查看情况，只能干等着，这时候出现了变数，如果超出了他们的控制范围，那事情可就大条了。

在众人的议论声中，少女的体型逐渐回归正常，甚至身上的衣服也能见到逐渐变干，这个速度实在是超越了常人所能理解的速度(这里衣服变干的速度大概就跟游戏里的潮湿状态消失的速度一样)，忽然，在少女身体的上空，出现了一串大字。

"Hello World"

 人们惊慌了起来，无论是懂元素力的，还是不懂元素力的，都开始产生了恐慌，因为这种样式的文字出现，是绝对不可能出现非橘红色的颜色，这种蓝色的颜色，带着一丝清丽和诡异，将在场的所有人的世界观给崩碎。

当然了，世界观既然碎了，那就得碎的彻底一点。于是少女动了起来，诈尸一样的坐起身，掀掉了脸上的白布，看着眼前的人群闪过了一丝迷茫，众人呆呆的看着少女慢悠悠的起身，拍了拍离她最近的千岩军，愣了两秒，又忽然跳起，仿佛空中飞人一般很快消失了踪迹，空空如也的白布仿佛告诉他们盖了个寂寞，而千岩军和围观群众也一脸懵逼的宕机在当场。

“卧槽，TM诈尸了！？”终于有个心大的回过神来，惊慌的喊道，他的喊声有如撒入了热水中的镁粉一般，恍然炸开了锅，众人惊慌的想要逃离现场，千岩军们虽然也心中发慌，但是为了尽快疏散人群，他们还是快速出动，维持着现场的秩序，不停的安抚着他们。这件事铁定是盖不住了，不如早点疏散好人群，让人直接通报七星才是良策。

璃月岗的动乱显然还没这么快传到云溪这里，他还在和他父母美美的吃着饭，母亲也很高兴，显然没有看到纪存初和云溪对话的她，心中只有对云溪未来的殷殷期盼，不停的嘱咐着云溪未来和古华派的人打好关系，为自己的人生铺路云云，只有云老爹有些兴致怏怏的扒着饭。

忽然，云溪心中闪过一丝悸动，看向了房顶，但这种感觉稍纵即逝，云溪奇怪的摸摸鼻子，继续吃饭。要做的事情还有很多，目前的人生也暂时开始明朗，未来几年的自己也注定无法闲着，他必须为了自己做一个更好的打算了。

。。。

纪存初回到了家，老婆子确实让人做好了饭菜，纪芾看到了纪存初回来，赶忙过来搭把手，而纪存初也直接说道：“阿芾，你的元素力练习的怎么样了？”

纪芾一愣，自从年前觉醒元素力之后，纪存初每个月初都会来问一下进展，但上周刚看过自己的元素力情况了，今天怎么又问了一下？

不过纪芾对于爷爷有着发自心底的崇敬，连忙回到：“爷爷，我现在已经能熟练使用类语句块儿的咒语了。需要我向您展示一下成果么？”

见到纪存初点点头，米芾刚准备催动，却听见纪芾奶奶催道：“先吃饭，什么事情等到吃完饭再说，你们爷俩也不看看现在几点了，菜都凉了。”

纪存初怔了一下，恢复了正常的神态，哈哈大笑一声：“好，好，先吃饭，吃完阿芾你可要好好给爷爷我露两手。”

不过爷俩兴致倒确实都不在吃饭上了，草草吃完了饭，纪芾就在院子里开始催动了咒语。

using System;
public class Demo{
    public static void Main(string\[\] args){
        string move = "乌鸦坐飞机";
        Console.WriteLine(move);
    }
}

很快一股轻微的压力从纪芾头顶逸散，而且天空中也飘出了几个字符

"乌鸦坐飞机"

纪存初刚想夸上两口，嘴角又抽了抽：“阿芾，怎么？又是自己想出来的奇怪的招式名？”

纪芾挠挠头，憨笑了下：“也是当时看闲书得到的灵感，感觉这句话朗朗上口，就当作这名字了。”

“好，很不错”纪存初拍拍手，也不知道是夸赞的纪芾的咒语的熟悉程度，还是他的起名能力，“不过，阿芾，你要知道，有时候咒语的显示文字并不是唯一催动元素力的结果，比如这样。”

米芾刚消化爷爷说的这句话，忽然一股同样轻微的压力从头顶上压下，爷爷腰间的神之眼同样像刚才自己催动的那样，呈现一种瑰丽的橙黄色，但是爷爷身前却没有显示字符。这样米芾很是惊奇，一个咒语变量被定义后，如果不使用的话，它就不会触发任何元素力的效果，只会在神之眼上出现对应元素力颜色标志，但爷爷仿佛可以直接催动变量的效果一样，他急忙问道：“爷爷，这是怎么做到的？”

纪存初笑而不语，沉思了一下，一串字符从身边显现

@"using System;
public class Demo{
    public static void Main(string\[\] args){
        string move = "乌鸦坐飞机";
        string a = move;
    }
}"

纪芾心领神会，试着默念了一下，果然身边有着可有可无的压力传来，但却没有字符显示，纪存初此时突然问道：“阿芾，你知道是什么原因么？”

纪芾一愣，随即皱着眉头思索了起来，以一种不确定的语气说道：“因为这给变量被调用过了，只是。。。不是以字符显示的形式？”

“是，也不是。”纪存初突然像老神棍儿一样的笑起来，“阿芾，你继续往下学吧，等你学到了后面，自然会明白的。
