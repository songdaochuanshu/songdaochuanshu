2023年12月16日-17日，一年一度的 .NET Conf China 2023 中国 .NET 开发者大会在北京盛大举办！大会以第一天主会场 + AI、.NET 8、云原生、IoT、前端&安全5大分会场，第二天闭门会议 + 三场工作坊的形式，为现场以及线上的 .NET 爱好者们带来了为期两天的 .NET 年度盛宴。

本次 .NET Conf China 2023 大会由 .NET 中文社区主办，微软 Reactor、智用人工智能应用研究院、基普智能、盛派网络、微软最有价值专家、米立科技共同协办。

本次大会共有30余名 .NET 各技术领域的专家带来主题分享，近300名观众到场参会，15家社区及企业在现场开设展位，线上/线下20余名志愿者参与支持。大会同时在线直播，超过16万人次观众在线观看了本次大会直播。

**工作坊**

**领域驱动设计从战略到战术 - .NET 也许是目前最适合领域驱动设计落地的技术栈**

FireUG 技术社区组织者之一肖伟宇带来工作坊：领域驱动设计从战略到战术 - .NET 也许是目前最适合领域驱动设计落地的技术栈，带大家了解“领域驱动设计”第一性原理，体验 .NET 加持下的领域驱动设计实践。
