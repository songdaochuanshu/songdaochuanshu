TIOBE 12月榜单: C# 即将成为2023 年度编程语言
===============================

TIOBE 公布了 2023 年 12 月的[编程语言排行榜](https://www.tiobe.com/tiobe-index/)。 2022年C# 在挑战成为年度编程语言，但在最后一刻，C++出人意料地夺得了冠军。今年，我们确信 C# 将获胜成为2023年度编程语言。它在 1 年内上涨了 2.38%，而其最接近的竞争者 Fortran 和 F# 分别仅上涨了 +0.64% 和 +0.48%。[TIOBE的10月份榜单](https://www.oschina.net/news/260951/tiobe-index-2023010)，C# 和 Java 之间的差距越来越小，达到了历史新低，10月份两者的差距仅为 1.2%。 这一趋势在12月继续保持， 目前两者差距缩小到了0.69%。

C# 相比Java，最大的优势在于，可以非常高效的用在工业界，对接大量已存在的C, C++代码, 但是Java就不行。特别是.NET 8 的NativeAOT的支持越发完善。 C# 十分接近于传统高级语言 C 和 C++，是一门面向对象的编程语言，但是它与 Java 非常相似，有许多强大的编程功能，因此得到广大程序员的青睐.

2023年12月16日在北京线下举办为期两天的 .NET Conf Local Event ，邀请中国开发人员共襄盛举。今年峰会的主题是 -- “**智能 开源 安全**”。 第一天主会场上开放原子开源基金会旗下 openEuler 社区 将分享《openEuler 社区对.NET 的兼容支持介绍》以及 龙芯.NET 团队的《LoongArch64 架构对.NET 扩展特性支持情况》，

近年来，开源发展模式越来越深入人心，我们迫切需要重视开源、学习开源和运用开源，成为中国融入科技创新网络，参与国际科技治理的有效途径。
