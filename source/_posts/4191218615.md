1、初始化项目
=======

`git init`  
初始化项目，先在本地去创建git仓库（后面才同步到关联的远程仓库托管）  
![在这里插入图片描述](https://img2022.cnblogs.com/blog/2420764/202205/2420764-20220510200255745-106246471.png)

2、将当前目录下所有需要上传的文件代码等资源添加到缓存区
============================

`git add .`  
![在这里插入图片描述](https://img2022.cnblogs.com/blog/2420764/202205/2420764-20220510200255850-226067520.png)

3、 提交缓存区里面的主要内容到本地仓库
====================

`git commit -m "提交说明"`

![在这里插入图片描述](https://img2022.cnblogs.com/blog/2420764/202205/2420764-20220510200256365-1880048109.png)

4、添加一个远程仓库的地址
=============

`git remote add origin https://gitee.com/zhang-bingqian/crm.git`  
（远程仓库的地址）

![在这里插入图片描述](https://img2022.cnblogs.com/blog/2420764/202205/2420764-20220510200255744-1493349356.png)

5、提交本地仓库到远程仓库
=============

`git push -u origin master`  
因为第一次提交 所以要加一个 -u

![在这里插入图片描述](https://img2022.cnblogs.com/blog/2420764/202205/2420764-20220510200255676-729369126.png)
