抢先体验! 在浏览器里写 Flutter 是一种什么体验？
-----------------------------

![](https://devrel.andfun.cn/devrel/posts/2022/07/RlDMvo.jpg)

Invertase 是一间位于英国的开源软件制作公司。主要构建关于开发者工具、SDK 等应用程序，早在 [Flutter 2.2](https://flutter.cn/posts/whats-new-in-flutter-2-2) 的时候，Invertase 团队就开始帮助构建和贡献 Firebase 相关的 Flutter 插件 (FlutterFire)；在 [Flutter 2.8](https://flutter.cn/posts/whats-new-in-flutter-2-8) 正式版发布的时候，文章里也介绍了他们和 Canonical 的合作：共同在 Linux 和 Windows 上开发最流行的 Flutter Firebase 插件。除此之外，他们还构建了很多开源产品，你可以在他们的 GitHub 主页上查看：[github.com/invertase](https://github.com/invertase "Invertase 团队的 GitHub 主页")

昨天，Invertase 团队发布了一款新的产品「Zapp」，一个在浏览器里构建 Flutter 应用的 Web 应用：

Zapp 服务支持使用任何 Pub 上的 package，也通过内置的 Dart 分析器提供语法高亮和自动完成等功能，与此同时也可以将你的工程代码分享给其他人，也可以像 VS Code 中一样使用命令面板执行操作，还有深色主题、调试、Git 代码管理、测试等功能。

![](https://devrel.andfun.cn/devrel/posts/2022/07/6ce5af77c7ec8.gif)

_配图来自 [Zapp 的命令面板功能](**b**/blog/announcing-zapp-rapid-flutter-dart-development-environment-in-browser "Zapp 官宣文章: Announcing Zapp, an in browser sandbox environment to build Flutter apps")_

目前 Zapp 正处于封闭测试期，任何人都可以申请加入 Zapp 的 EAP 计划，方法是打开 [应用官网：zapp.run](https://zapp.run "Zapp 应用主页")，并使用 GitHub 账户登陆，当收到 EAP 资格后，会收到电子邮件通知。

有任何功能反馈可以在 Zapp 的 [GitHub Issue 区](https://github.com/invertase/zapp.run/issues "Zapp 的产品反馈建议链接") 提出，期待社区成员们的试用报告和反馈！
