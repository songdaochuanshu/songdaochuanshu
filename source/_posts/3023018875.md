HMS Core图形图像技术展现最新功能和应用场景，加速构建数智生活
==================================

【2022年7月15日，杭州】HUAWEI Developer Day（华为开发者日，简称HDD）杭州站拉开帷幕。在数字经济不断发展的今天，开发者对图形图像的开发需求更加深入和多样化，从虚拟环境重构到用户交互再到视觉呈现，开发者对于数字世界存在一个完整的需求链条。在HMS Core.Sparkle应用创新分论坛中，HMS Core技术和产品专家与国内行业内开发者共同探讨图形技术的现状与应用，华为为开发者展示了最新的HMS Core [3D Engine](https://developer.huawei.com/consumer/cn/hms/3d-engine/)功能、[AR Engine](https://developer.huawei.com/consumer/cn/hms/huawei-arengine/)在淘宝极有家的应用，VR Engine、3D建模服务等，共同探索数字世界的移动应用发展之路。

![](https://img2022.cnblogs.com/other/2396482/202207/2396482-20220715180447813-77817142.png)

### 构建三维数字世界的基石

作为构建三维数字世界的基石，HMS Core 3D Engine提供了高性能、高画质、高扩展性的实时3D引擎，和便捷高效的可视化开发工具，帮助开发者制作高品质的3D应用。

在3D渲染方面，支持实时全局光照、动态多光源，并提供景深、环境光遮蔽、抗锯齿等后处理功能，高度还原逼真的光影效果；多维粒子系统支持CPU/GPU粒子渲染，可模拟爆炸、火焰、雨雪等等自然场景，及刀光剑影等特殊效果；实时骨骼动画、表情动画、动画重定向、多重动画状态机融合，可实现流畅的3D动画效果。

3D Engine不仅可以助力制作卡通虚拟数字人、手语写实数字人和虚拟舞台效果，还和超图一起推出了全新的高保真三维GIS插件，通过3D渲染技术助力三维GIS实现更高清、更具真实感的数字世界场景。

### 真实与虚拟无缝衔接

HMS Core在AR（增强现实）和VR（虚拟现实）上的技术深耕，也为数字世界的构建添砖加瓦。

HMS Core AR Engine提供虚实融合的关键算法，解决了坐标统一、场景统一、相机统一这3个问题，让虚拟物体可以实景放置。在运动跟踪、环境跟踪、人体和人脸跟踪等助力虚实融合的AR能力上，技术指标均有提高。淘宝极有家集成AR Engine后，拥有了AR测量功能，利用高精度环境SLAM技术，手机扫描即可轻松构建房屋环境。截止2022年6月，AR Engine已成为全球TOP3移动设备AR开发平台，有17亿次的安装量、2800+款的活跃三方应用数、120+款已覆盖机型。

![](https://img2022.cnblogs.com/other/2396482/202207/2396482-20220715180449380-1912542757.jpg)

HMS Core [VR Engine](https://developer.huawei.com/consumer/cn/vr)则和XR设备、Harmony OS、内容平台一起，打造XR世界生态基础。VR Engine是为华为VR内容开发者提供的平台，开发者使用华为VR SDK进行开发，开发完成后上传至华为VR应用商店，审核通过后拥有华为VR眼镜的消费者即可直接购买下载应用内容。HUAWEI VR目前主要的硬件形态是VR眼镜加华为系列手机，VR眼镜连接手机后，手机VR应用通过集成的VR Engine运行。比如，集成了VR Engine的华为VR Glass上市后屡获国内外大奖，用户通过一副眼镜即可连接不同产品，并且还可跨平台支持手机、PC、云。

### 助力3D动画制作和手语无障碍沟通

在3D动画制作方面，HMS Core [3D建模服务](https://developer.huawei.com/consumer/cn/hms/huawei-3d-modeling)新开放了自动骨骼绑定功能，即基于AI技术，可为二足人形3D模型赋予运动能力。自动骨骼绑定功能有高度自动化、超强鲁棒性、优异的蒙皮效果等优点，如通过普通手机拍摄二足人形物体多张图像实现高精度建模；自动化骨骼绑定，无需人工干预，通过业界主流引擎，即可实现一键驱动。若3D建模服务结合AR，则可以解锁更多玩法，如让模型走入真实场景，将电子宠物带到你的沙发、床等中来。

![](https://img2022.cnblogs.com/other/2396482/202207/2396482-20220715180452614-487440195.gif)

在手语数字人方面，在增强手控动作的基础上，HMS Core[手语服务](https://developer.huawei.com/consumer/cn/hms/signpal-kit/)赋予了手语数字人非手控特征，包括姿态、面部表情和口型，更加贴近真人手语动作，显著提升了手语整体可懂度。当前，手语服务面向媒体内容制作、教育学习和日常交流等高频场景，开放了移动端手语生成、文本转视频文件、手语直播等服务。

随着构建3D数字世界重要一环的图形技术的发展，虚拟世界与现实世界之间的界限将逐渐模糊，HMS Core助力各位开发者走好未来应用发展之路，赋能更多应用走入数字世界，共同促进数字生态的繁荣。
