HuggingChat macOS 版现已发布
=======================

Hugging Face 的开源聊天应用程序 Hugging Chat，现已推出适用于 macOS 的版本。

![](https://img2024.cnblogs.com/blog/3075972/202409/3075972-20240926202832589-1288422315.png)

主要特点
----

Hugging Chat macOS 版本具有以下亮点:

*   **强大的模型支持**: 用户可以一键访问多个顶尖的开源大语言模型，包括 Qwen 2.5 72B、Command R+、Phi 3.5、Mistral 12B 等等；
*   **丰富的功能**: 应用内置网络搜索和代码高亮等实用功能，未来还将推出更多增强功能；
*   **完全免费**: 所有功能均可免费使用，无需付费；
*   **精心设计的用户界面**: 我们为 Mac 用户量身打造了直观易用的界面，确保流畅的使用体验。
*   **通过应用快捷键呼出**: Hugging Chat 应用快捷键是 ⌘ + Shift + Return

参与测试
----

Hugging Chat macOS 应用目前处于 beta 测试阶段 (v0.5.0)，我们诚邀社区成员参与测试并提供宝贵反馈。主要测试内容包括:

*   登录和引导流程
*   聊天功能 (模型选择、响应生成、Markdown 解析等)
*   自定义选项 (代码高亮、外观设置、系统主题等)
*   性能表现
