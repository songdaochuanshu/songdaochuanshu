第二回 巧习得元素分类 子不知怀璧其罪

\=====================================================================

云溪父亲见状看了看云溪，脸上渗出意思冷汗，但迟疑一下就立即退了出去，匆匆忙忙的往右边厨房赶，只留下了云溪和这位神秘的老爷子。

云溪瞠目结舌的看着悠然自得的喝着老爹泡的茶的老爷子，下意识说了一句：“老先生你怎么这么快，还知道我要来这里”。

“方向，你一直在绕巷子，大致上方向却没变哦，而且我以前见过你。”老爷子又细嗦一口茶，看着云溪，指了指：“而且我还要讨要一下你手上的《原素要旨》。”

云溪这才注意到手上还攥着被汗水些许沾湿的书，立即向前恭敬地呈上：“十分抱歉，刚才小子过于慌乱了，这本书还给您。”

老爷子放下茶盏，将书推回云溪胸前，神秘兮兮的说到：“书只是数张白纸，没什么用，与其说我想要拿这本书，不如说我想要。。。你这个人。”

云溪父亲刚吩咐老婆做顿饭菜，刚回到门口，便见到如此诡异的一幕，咽了口口水，忙喊道：“纪大人真的是爱开玩笑啊，犬子哪里能入得了纪大人的法眼呢？”他说着快速走到云溪身边将云溪拉远一点点，对着纪老爷子做出请坐的手势，心中冷汗直冒，甚至有些后悔直接给钱让云溪自己买东西作为成人礼了，也不至于会闹得如此荒唐的事情来。、

这位可怜的老父亲，拼命的向着纪老爷子示好，而纪老爷子也看出来了这位的心思，哈哈笑一声：“云先生，你怕是不知道你家好小子今天在大庭广众下做了什么惊天动地的事情哦。”还不等云溪父亲脸色骤变，他又继续说道，“放心，不是什么坏事，是你家云溪澈成功的念出来了《原素要旨》的咒语，激活元素力了。”

云溪父亲宕机在当场，这个或许算得上很是令人高兴的事情却在着戏剧性的演化下变得让他有点手足无措，他看看低头不语的云溪，又看看眯眯眼笑着的纪老爷子，强行让自己镇定下来，笑着作揖：“那想必犬子激活元素力的时候纪大人也是在场吧，拿不知道纪大人是要对犬子做什么呢？如果需要我帮助的话尽管吩咐”他擦了擦汉，“对了，我让我堂客做了些饭菜，纪大人光临寒舍，还望由云某招待一番以表心意。”

纪老爷子摆摆手：“没什么事儿，不需要如此隆重，我刚才就已经说明了，就是来问着小子几个问题的，谁知道这小子谎成这个模样。对了，这个可能会触及一些隐私，所以。。？”、

云溪父亲知会了这个意思，担心的看了云溪一眼，立即悄悄退出去，并且将门半掩起来。等到脚步声确定远了，纪老爷子才像是发现了给新奇的宝贝一般端详起云溪来，不停的在绕着云溪转圈，而云溪内心也是越来越煎熬。

“我还没自我介绍呢想起来。”纪老爷子突然顿住身子，停了下来，“我是万文集舍的二级舍长，目前算是打理万文集舍的唯一负责人。今天早上我正在整理旧书集，结果楼下突然传出一股岩元素力，我一看，居然是你。”

居然？云溪一愣，着仿佛就像是再说阔别已久的老友忽然再现一般，让他很是费解：“小子。。。小子不记得见过纪大人。”

“什么纪大人纪大人的，生疏。我叫纪存初，你喊我纪伯就行，不然就显得太生分了。”纪存初皱了下眉头，拍了拍云溪，把云溪拍的冷汗直冒，他仔细看了看云溪脸色的汗，疑惑道，“我有这么可怕么，这么怕我？”

纪存初走回座位，又抿了一口茶：“好茶好茶，你爹品味还真挺不错。对了，你使用元素力时编译的咒语是什么？是书上那个？”

云溪看了一眼纪存初又赶紧低下头，思索了一阵，才回到：“是的，我完全按照书上的那个咒语来的。”

“书是好书，你可以完全参照它继续钻研下去，它经过了数千百年的编写，终究是能胜任大部分的‘教材’了，是一个最最权威的‘官方文档’也不为过”纪存初点点头，仔细的看向云溪，“不过也不可以全然照搬这上面的东西，否则就只会它写一你只会一，这可不是一个好的原神能做出来的蠢事。”

“小子。。小子斗胆问一句，这本书。。。”云溪呈起手中的书，纪存初已经看出来了他的心思，“拿去拿去，这又不是什么绝世秘诀，要不是销量不好，受众太少，当个课堂教科书都行，我以前每年印那么一批，来年都不一定能卖出去10本，你这手上的还是我10年前印的，与其当废纸还不如给你当宝贝呢。”

云溪咽了下口水，赶忙准备将书收起来，却又听纪存初喊道：“哎哎哎，别这么急啊，我还没问完呢，现在的小伙子怎么都毛毛躁躁的。”

“小子，你把你早上使用的咒语再施展一遍？”

云溪应声点点头，翻开书找到了第一页，照着书上的开始心中默念了起来

class Example{
    static void Main(){
        System.Console.WriteLine("我是我");
    }
}

很快便如同所预料的那样，空中开始显示出了对应的文字来，云溪这时候才有机会好好观赏一下他创造出来的东西，三个字充满着橙黄甚至带点火红的颜色，静悄悄的就这么展示在空中。纪存初显然遇到过很多个这种情况了，很平常的看了一眼，皱了皱眉头：“你确定这是和早上使用的咒语是一样的么？”

“纪。。纪伯，是一样的。”云溪恭恭敬敬的回答，纪存初突然冷下来的脸让他又开始紧张了起来。

“唉，行吧。”纪存初薅了薅有些杂乱的白发，想到了什么，稍微沉吟一下，一段文字就这么在他的眼前跳了出来，在文字跳出来的一瞬间，云溪甚至感到身边有一阵细微的轻风拂过。

"using System;
class Example{
    static void Main(){
        Console.WriteLine("我是我");
    }
}"

显然这段文字是纪存初创造出来的，纪存初确认了下内容，对云溪说到：“快，你再按照这个咒语做一遍”。

云溪显然照做了，但是让他疑惑的是，默念照着样子默想完咒语后，什么也没发生，纪存初也大为诧异，不过在一秒后他想到了什么，往云溪身上重重一拍，云溪身上立即浮现了几个大字

CS0201

“草”显然这位纪老爷子的脾气并不是那种和善型的，开口就是一句优美的璃月话，他看向云溪，“哎呀，不是让你带着引号一起默念啦，只是执行里面的东西而已。”、

云溪尴尬的挠挠头，立马开始冥想引号内的咒语，这回倒是和预想般的那样，再次出现了一行字

"我是我"

但是他又疑惑起来，这两个咒语好像涨的并不一样，但出现的效果好像是相同的，甚至自己的身体上都感觉不出什么实在的区别来。他下意识拿起书本，但繁多的内容里显然无法一下子就找到自己想要的结果。

虽然这次的结果很是正常了，但纪老爷子却好像更不开心了一样，皱着眉头在那儿愣神思索，云溪见纪存初在那里入了神，悄悄退了两步，借着窗子的光继续看下去。

书上的例子显然只是一个新手引导一般的东西，在看完这篇引言后，他终于看到了第一章：类型。

“正如光有着七种色彩一样，元素力也有多种多样的类型存在，每个人理论上都能使用任何一种元素力，并且在对应的神之眼上会出现不同的颜色来，达到区别于神之眼图案的另一种神之眼奇像。经过数百年的研究，终于能确定具体的元素力是通过什么类型才能激发出来，目前已知的咒语类型为

byte,sbyte,ushort,short,char,uint,int,ulong,long,decimal,float,double,bool

，在使用该咒语的时候，需要为这个咒语的量词规定一个命名，我们将之称作‘定义’，常见的定义咒语如下：

int num = 0;
//类型 变量名 = 变量的值;(分号用以保证该语句结束，是一个完整的语句)

”

依然是懵逼的一天，好在神秘的好心人(?)夏普贴心的给出了更进和解释。

“这本书怎么几百年都不更新迭代一下，这样会让很多人犯迷糊的，我在这里着重解释一下，在最新的几十年的研究内，发现还有一种混合这些类型的类型，这些特殊类型无论怎么变换，只要不换到上面说的那些类型，都会使神之眼呈现一种乳白色的光泽，具体的白色纯度还未有定数，还有一种特殊的，也是在前面引言内没有被古代科学家解决的问题，便是‘我是我’，这个本身也有类型，现在已经被研究出来，这个类型的对应咒语叫做

string

，这个虽然被纳入了特殊object中，但是它生成的元素力光芒仍然是和char，short一样的橙黄色光芒，目前的主流理论则是认为，它是char的一种变体，或者说是一个包含着许多char的集合。目前来说补充就这么多。还望诸君多多实践一下。”

云溪赶忙合上书，冥想了下刚才书上的定义int变量的咒语，另他奇怪的是，好像什么都没有发生，也没有文字，也没有画面，唯一不同的是，神之眼亮了，在他冥想的那一下，神之眼闪过了一丝草绿色，又忽然不见，他手撑着下巴，快速思索着，忽然，他的脑子里产生了一个大胆的想法。

int num = 0;
Console.WriteLine(num);

“嘶——”想象中的成功的生成并没有成功，反而闪过了一丝胸闷的感觉，眼中仿佛闪过了一个数字，如果看的没错，应该是"CS0103"。云溪长呼口气，虽然不知道这是什么意思，但显然不是他目前想要的结果，于是他想了想，再次试了试

int num = 0;
System.Console.WriteLine(num);

看见草绿色的颜色在眼前亮起，并且成功的展现出0的时候，云溪高兴的快要跳起来了，这就像是被泡在水缸里不得呼吸，最后浮出水面呼吸到了清爽的空气一样，这是一种从灵魂深处赐予的爽快感。他再次想了想，又冥想出另一个咒语来。

using System;
int num = 0;
Console.WriteLine(num);

显然，这次也很成功，虽然目前他还不怎么理解这里面的原理，但这并不会妨碍自己获得这种凭空创造的快乐。

“好玩吧，没玩过吧？”忽然一个声音响起，让云溪立马回到了现实，纪存初笑着半倚在桌子边儿，啧啧称奇：“我倒是有见过一天能按照书上的内容创造成功结果的，但是在短时间内就会玩儿这么多花儿的，啧啧，你独一个啊。”纪存初说着起身逼向云溪旁边，看着高了自己小半个头的云溪说到：“以前用过元素力？”

云溪害怕又疑惑的看着纪存初，咽了口口水，挠挠头，“纪伯，我今天是第一次用元素力。这才是仅仅会显示一些字而已。”

纪存初哈哈笑了两声，盯着云溪，云溪被盯得有些发毛，眼神垂了下去，纪存初又自顾自的在云溪面前踱步：“当一个人会使用神之眼操作神之力的时候，他便有了成神的资格，虽然目前千万年来没有见到过一个活生生的人成神，甚至没有出现过哪怕本就存在的神明过，但本着人对于神的向往和崇拜，人们还是将这种有着成神希望的人称作：‘原神’。这是自古就有的概念，同样的，就像是圣洁的神必定会有相伴的魔一样，有一个东西，即使存在于传说中，却仍然是被人谈之色变的存在。”

纪存初顿住脚步，凝重又狂热的死死盯着云溪：“云溪，你。。。见过黑色的神之眼么？”
