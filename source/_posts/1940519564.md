微软在Microsoft Build 2024 上 发布了.NET 9 预览版4
========================================

[![image](https://img2023.cnblogs.com/blog/510/202405/510-20240527075632221-1259306936.png "image")](https://img2023.cnblogs.com/blog/510/202405/510-20240527075629614-516680492.png)

在 [Microsoft Build 2024](https://build.microsoft.com/) 上，与往年一样，.NET 不是会议主题演讲的主题，但是微软在这个大会上为.NET 推出一组新的功能和工具，旨在使 .NET 开发更快、更轻松,具体内容可以阅读文章：[Microsoft Build 2024 的 .NET 公告和更新](https://devblogs.microsoft.com/dotnet/dotnet-build-2024-announcements/)\[1\]。最新功能都在.NET 9 Preview 4, 该更新旨在通过性能提升和优化增强来改善对 .NET 的支持。Microsoft说，关键是改善生成式人工智能应用程序以及基于云的Web应用程序的应用程序构建体验。这要归功于运行时、库、SDK（软件开发工具包）、.NET 多平台应用 UI （MAUI）、.NET Aspire、Entity Framework Core （EF Core） 等方面的“多项优化和增强”

[![image](https://img2023.cnblogs.com/blog/510/202405/510-20240527075637200-989029144.png "image")](https://img2023.cnblogs.com/blog/510/202405/510-20240527075634945-1698773176.png)

其中.NET Aspire已经正式发布-[.NET Aspire 正式发布：简化 .NET 云原生开发](https://www.cnblogs.com/shanyou/p/18205302)\[2\]，在[用关键词“.NET”](https://build.microsoft.com/en-US/sessions?search=.net+&sortBy=relevance) \[3\]查找Build2024 的相关主题，你将找到一些实时会话，例如，周四下午 5：30 CEST [的 C# 13.0](https://build.microsoft.com/en-US/sessions/689e5104-72e9-4d02-bb52-77676d1ec5bc?source=sessions) \[4\]和 [Entity Framework Core 的 MongoDB 数据库驱动程序](https://build.microsoft.com/en-US/sessions/b5cfd926-3b70-4b46-b6b8-5532413fe8b1?source=sessions)\[5\]（MongoDB Provider for Entity Framework Core 现已正式发布。该功能于去年 2023 年 10 月宣布），以及以前录制的有关 [ASP.NET Core 9.0 和 Blazor 9.0](https://build.microsoft.com/en-US/sessions/6f3ea491-a05d-4693-bfce-60361e230ade?source=sessions) \[6\]以及 [Entity Framework Core 9.0](https://build.microsoft.com/en-US/sessions/4f008f3d-c58a-4806-b6b4-dfb993b32c4f?source=sessions) \[7\]的相关主题。
