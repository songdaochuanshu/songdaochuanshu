2\. GitHub Trending 周榜
----------------------

### 2.1 Follina 攻击：msdt-follina

**本周 star 增长数：950+**

`New` msdt-follina 使用恶意 Microsoft Word 文档创建一个 Follina MS-MSDT 攻击，并对 HTTP 服务器进行有效负载。Follina 是近日被发现的高危 0day 漏洞，可基于 Word 权限执行任意代码，变更、删除资料，以及创建新账号。

> GitHub 地址→[https://github.com/JohnHammond/msdt-follina](https://github.com/JohnHammond/msdt-follina)

![](https://img2022.cnblogs.com/blog/759200/202206/759200-20220605185641221-2119518778.png)

### 2.2 在 GitHub 仓库记笔记：gitnoter

**本周 star 增长数：900+**

`New` GitNoter 是一个 Web 应用，允许用户在自己的 git 仓库中存储笔记。特性：

*   GitHub 登录
*   高颜值的操作界面
*   支持 Markdown 语法
*   支持 Markdown 预览
*   剪切板功能可快速拷贝代码段
*   直接在根目录存储笔记
*   笔记设有缓存以避免额外的 API 调用
*   urls 可添加书签

> GitHub 地址→[https://github.com/git-noter/gitnoter](https://github.com/git-noter/gitnoter)

![](https://img2022.cnblogs.com/blog/759200/202206/759200-20220605185701414-147623428.gif)

### 2.3 因果推理 Python 库：DoWhy

**本周 star 增长数 500+**

DoWhy 是一个因果推理的 Python 库，支持对因果假设的显式建模和测试。DoWhy 基于因果推理的统一语言，结合了因果图5模型和潜在结果框架。

> GitHub 地址→[https://github.com/py-why/dowhy](https://github.com/py-why/dowhy)

![](https://img2022.cnblogs.com/blog/759200/202206/759200-20220605185711162-1281287889.png)

### 2.4 服务启动页：dashy

**本周 star 增长数：550+**

`New` dashy 是一个服务自托管启动页，它可方便让你通过图标访问对应应用，也可以通过搜索等常规方式找寻应用。它可将你所有同防火墙、DNS 等等相关的服务汇集在一处，方便你访问。最重要的是，这大概是我见过最好看的服务管理工具了。

> GitHub 地址→[https://github.com/Lissy93/dashy](https://github.com/Lissy93/dashy)

![](https://img2022.cnblogs.com/blog/759200/202206/759200-20220605185720440-2072333982.gif)

`New` DeepFaceLive 支持你在 流媒体/视频 通话时实时换脸。
