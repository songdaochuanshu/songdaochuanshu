电脑微信中快捷键的秘密——聊聊那些你知道的和不知道的微信快捷键
===============================

微信中你熟悉的全局快捷键和你不熟悉的局部快捷键介绍。

微信是我们在熟悉不过的沟通工具了。往往出于各种原因的需要，我们需要在电脑端使用微信收发消息。怎样提高微信的使用效率，就至关重要了。本文正是出于此目的，笔者写给视障小伙伴的微信经验谈。本期的主题——微信中那些快捷键的秘密。

一提到快捷键，有的小伙伴可能会嘀咕了：“是不是很难记？”

但我想，比起记忆这点儿快捷键的痛苦来说，效率提升会让你感受到快乐的。

微信快捷键会分两类。一类是**全局快捷键**，及无论是在微信界面下，还是在别的软件界面下，这些快捷键都可以使用。另一种是**局部快捷键**则需要在特定的微信界面下才会有相应的功能。

显示/隐藏微信主界面
----------

如果你需要快速显示出微信的主界面，显示/隐藏微信主界面的快捷键就很有用了。默认情况下，你可以按下 **CTRL + ALT + W** 来使用它。

微信截屏工具
------

如果你需要截图的话，微信也提供了一个截图工具，可以截全屏，也可以截取某个窗口。但这个功能对于屏幕阅读器用户是没价值的，默认快捷键是 **ALT + A**。如果这个快捷键影响到了你使用其他软件，笔者建议你取消它。

全局快捷键的修改
--------

如果以上两个快捷键你觉得不顺手，或者你压根不需要，就需要对其进行修改，方法是：

1.  在威信主界面，点击导航栏中的“更多”按钮，接着在弹出的菜单中选择“设置”，打开微信的设置界面。
2.  在微信的“设置”界面，切换到“快捷键”选项卡。
3.  点击需要修改的快捷键。对于键盘用户，则用 Tab 键切换到需要修改的快捷键按钮上，点击空格进入修改快捷键状态。
4.  按下你想要设置的快捷键。按下退格键则会清除快捷键。
5.  点击“确定”按钮完成修改。对于键盘用户，则需要先 Tab 键**切换到**“确定”按钮，接着按下**回车**完成修改。

如果改错了，也可以先点击“恢复默认设置”，再尝试修改。

聪明的你，可能在“快捷键”设置界面发现还有一个设置发送消息的快捷键。下面我们就来介绍一下。

发送消息的快捷键
--------

在“快捷键”设置中，提供了两个发送消息的快捷键共用户选择：

*   Enter：在消息输入框中，按下 Enter 回车键发送消息，按下 CTRL + Enter （回车键）换行；
*   CTRL + Enter：在消息输入框中，按下 CTRL + Enter （回车键）发送消息，按下 Enter （回车键换行

两种方式，可根据个人习惯任选其一。

另外，微信还有一个通用的发送消息的快捷键，下文就介绍给你哦。

微信中的局部快捷键
---------

每当我们召唤出微信主界面的时候，我们最希望的就是看一下哪些人/群给我们发了心消息。如何快速切换到“绘画”列表呢？

**ALT + L** 就可以快速将键盘焦点移至“绘画列表”。接着上下光标就可查看哪些人/群给我们发了消息了。如果要看具体的内容，我们有两种选择：

1.  按下空格键： 该人/群的绘画窗口，则会**平铺**在微信主界面的右侧区域；
2.  按下回车键： 该人/群的绘画窗口则以一个**独立**的窗口展示出来。

回到我们看消息上来，如果我们要快速找到消息列表怎么办呢？

**ALT + M** 这个快捷键，可将焦点移至消息列表。接着就可以上下光标浏览每一条消息了。当然，你还可以按下 Applications 键，对消息进行操作，如收藏、语音转文字等。

看玩了消息，就可能要回复对方消息了。如果此时你还没在消息输入框键入过内容，可使用 **ALT + S**，移至消息输入框，编写你要发送的消息。完成之后再次按下 **ALT + S** 就可以发送该条消息啦。

最后，可使用 **ALT + C** 关闭窗口。
