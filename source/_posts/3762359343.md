.NET Conf China 2023济南站社区活动
===========================

 2024年3月3日，在这个春暖花开的日子里，由微软MVP项目、山东财经大学管理科学与工程学院、胶东开发者社区、济南.NET俱乐部联合举办了【.NET Conf China 2023 JiNan Watch Party：走进山财大—AI驱动下的应用智能化开发】活动。

本次社区技术活动采用了高校、社区联合举办的模式，走进省属一流高校-山东财经大学，活动得到了山东财经大学管理科学与工程学院院长刘位龙教授、党委书记徐世星书记以及张戈、王华杰等多名老师的大力支持。

数十名来济南及省内其他城市的AI、.NET开发者和爱好者们齐聚山东财经大学，共同探讨交流AI驱动下的应用智能化开发。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190137020-937259631.png)

一、会议整体日程
========

在这个大数据与智能技术飞速发展的时代，我们见证了大模型和开源项目在推动智能应用创新方面的巨大潜力。本次会议聚焦于业界关注的焦点：LLM大语言模型、.NET技术、Semantic Kernal开源框架。结合实际的产品开发案例，探讨AI如何驱动应用智能化的发展，如何利用AIGC技术，实现应用程序的智能化改造与升级。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190147635-159246351.png)

本次活动设置了四大议题：

● 结合 Semantic Kernel 和 Kernel Memory 实现 AI Agent 的探索

● AIGC-基于大模型的智能翻译和数据分析

● AIGC-基于大模型及向量数据库为零售行业精准营销赋能

● .NET在云中的优势与云原生实践

本次活动邀请了省内AI和.NET领域的技术专家和社区领袖，参会人员总计31人，包含山东财经大学的教师、学生以及济南各企业和社区的技术开发者，第一次将最新的AI技术和智能应用覆盖到高校和技术社区，得到了参会者和学院领导的一致好评。

接下来我们回顾一下本次活动的主要内容。

二、学院领导致辞
========

首先，由山东财经大学管理科学与工程学院刘位龙院长发表开场致辞。刘院长热情洋溢的话语，为本次活动开启了精彩的序幕。他全面介绍了管理科学与工程学院的发展历史、办学特色、学科设置、产学研项目以及省级工程实验室、省级工程技术研究中心等情况。让我们每个参会人感受到学院在高等学校教育上的沉淀、发展和创新。

三、专项议题1-结合 Semantic Kernel和Kernel Memory 实现 Al Agent 的探索
========================================================

活动致辞之后，微软最有价值专家、基普智能创始人衣明志老师开启了第一场演讲，他以“结合 Semantic Kernel和Kernel Memory 实现 Al Agent 的探索”为主题，为在座的每一位听众展示了AI的无限可能。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190224687-334331015.png)

衣明志老师在本专题中，详细介绍了他个人及其团队近期在AI领域的一些创新：

● ai-shell 增加了12种语言的支持，基于自然语言的快速代码开发

● 一份普通的需求文档通过AI转换成符合该需求的系统接口

● 基于自然语言的非破坏性 B/S 系统智能外挂技术

同时跟社区的小伙伴深入分享了企业的数字化/智能化在AI时代的玩法，SK框架的技术原理以及基于SK框架如何构建AI Agents。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190239278-684308479.png)

四、专项议题2-AIGC-基于大模型的智能翻译和数据分析
============================

特来电云平台副总经理、微软最有价值专家周国庆老师带来了主题为“AIGC-基于大模型的智能翻译和数据分析”的精彩演讲。他深入浅出地介绍了大模型在智能翻译和数据分析中的广泛应用，引发了在场众多听众的热烈讨论。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190239067-1431871038.jpg)

周国庆老师在这个议题中，重点给大家分享了基于LLM大语言模型如果构建一个智能化的翻译服务和数据分析应用。同时以实际的案例给大家分享了提示工程。同时演示了使用Semantic Kernel开源框架，基于GPT4实现翻译服务, 用于专业翻译的真实业务案例。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190239489-1171460719.png)

五、  专项议题3-AIGC-基于大模型及向量数据库为零售行业精准营销赋能
=====================================

接下来，上海财经大学大数据方向博士、山东财经大学管理科学与工程学院教师、库米云创始人王华杰老师以“AIGC-基于大模型及向量数据库为零售行业精准营销赋能”为主题，为我们揭示了大数据在零售行业中的巨大潜力。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190239364-1635751598.png)

王华杰老师详细介绍了库米云相关的AI产品。利用联邦计算，构建安全多方计算，确保数据安全和隐私安全的前提下构建客户模型，同时根据模型生成客户行为预测，进行人群营销，对电商平台及线下门店的全域会员进行360会员画像及触达，提升全域全业态业绩，帮助零售企业完成数智化升级。

同时在AIGC应用方面，王华杰老师深入分享了LLM大模型、RAG检索增强生成、向量化(Embedding)等方向的技术研究和面向新零售的应用创新。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190239630-252142419.png)

六、  专项议题4-.NET在云中的优势与云原生实践
==========================

微软最具有价值专家冯辉老师的主题演讲“.NET在云中的优势与云原生实践”为我们展示了.NET在云原生领域的优势和实践经验。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190239130-639973906.jpg)

议题详细介绍了云原生应用与容器化简介、.NET对容器化的支持，以及.NET云原生实践。为广大.NET开发者在云原生应用的开发给出了大量一线实践方案。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190239643-1633371638.png)

七、  产学研结合-深入探讨学院课程设计和高校教育
=========================

在当今这个快速发展的时代，产学研结合已成为高等教育改革的重要方向之一。为了进一步强化这一理念，并将其落实到实际的教学活动中，刘院长和张戈老师联合举办了一场关于管理科学与工程学院课程设计的专项探讨会。这次讨论会不仅仅局限于学术界的教授和研究人员，还邀请了来自业界的专家和开发者，目的是为了打破传统教育与实际工作需求之间的鸿沟，更好地满足企业对于技术人才的实际需求。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190239172-738105574.jpg)

在这次会议中，我们首先审视了当前管理科学与工程领域的课程设置。刘院长开场强调，课程设计应当反映出行业内最前沿的技术趋势，并且要与时俱进。他提出了一个核心问题：“我们的课程，是否能够培养出能直接投入到企业技术创新中的高素质人才？”这个问题引发了与会者的深思。

随后，张戈老师展示了一份详尽的课程设计和规划，随着人工智能、大数据、云计算等技术的日益成熟，企业对于具备这些技术背景的研发人员的需求也在不断增长。因此，学院的课程设置需要紧跟技术发展的脚步，及时更新和调整课程内容，以确保学生能够掌握最新的技术知识。

会议的高潮部分是与会者分组进行的头脑风暴环节。各组成员结合自身的专业知识和实际工作经验，对课程设计提出了具体的建议。这些建议既包括了理论与实践相结合的课程模块，如将机器学习、网络安全等热门领域的最新研究成果融入到课程中，也包括了强化与企业合作的实践环节，比如增加实习机会、企业项目案例分析等。

在讨论的最后阶段，与会者共同确定了一套初步的课程改革方案。这套方案旨在确保学院的课程能够充分反映市场需求，同时也能够激发学生的创新精神和实践能力。为了实现这一目标，方案建议建立一个由教师、学生、企业代表和行业专家组成的持续性课程评估和更新机制。

通过这次深入的探讨，管理科学与工程学院的课程设计将更加贴近实际，更能够适应未来技术发展的需要。这不仅仅是一个教学改革的开始，更是学院与企业、社区合作的新篇章，预示着产学研结合的新时代即将到来。

八、  活动彩蛋-使用GPT4大模型进行活动抽奖
========================

在这次探讨会的尾声，主办方准备了一个特别的环节，以增加活动的互动性和趣味性——一个结合了创新技术的抽奖活动。刘院长宣布，这次抽奖将使用OpenAI开发的GPT-4大模型来进行，这不仅仅是一次简单的奖品分发，更是对人工智能在实际应用中潜力的一次生动展示。GPT-4被设定为抽奖的“主持人”。在活动开始时，通过一个界面与GPT-4交互，请求它随机挑选出幸运嘉宾。

![](https://img2024.cnblogs.com/blog/23525/202403/23525-20240309190239180-459165922.jpg)

整个抽奖环节不仅充满了欢声笑语，还让与会者深刻感受到了人工智能技术的魅力和潜力。通过这个环节，GPT-4的展示不仅增强了活动的趣味性，也让与会者对人工智能在未来教育和研究中的应用充满了期待。

随着抽奖环节的圆满结束，【.NET Conf China 2023 JiNan Watch Party：走进山财大—AI驱动下的应用智能化开发】活动也逐渐落下帷幕。在此，我们要特别感谢山东财经大学管理科学与工程学院的领导和老师们的无私支持和辛勤工作。院长刘位龙教授、党委书记徐世星书记、张戈老师、王华杰老师，您们的大力支持是本次活动能够成功举办的关键。感谢您们在百忙之中抽出宝贵的时间，为我们提供了宝贵的资源和支持。

【.NET Conf China 2023 JiNan Watch Party：走进山财大—AI驱动下的应用智能化开发】活动的成功举办，是所有人共同努力的结果。我们期待未来能有更多这样的机会，让产学研结合的理念在山东财经大学乃至更广阔的领域绽放光芒。再次感谢各位的支持和参与，让我们共同期待下一次的精彩相聚！
