“我们来玩双武将吧”“好啊，你是什么”“甄姬陆逊”
=========================

从来如此，便对么？

对门那家搬走了，来来回回一个礼拜，搬得一点不剩了。那些友人们，一个已经高二，一个参加工作，一个搬至别处，自己这个，也已初三了。想来，是没有机会玩三国杀了。

大我九岁那个，不妨将之称为小 X。这家伙从小被受压迫，因此高中叛逆，高考复读，是个可怜的人。我记得妈妈在花鸟市场给我买了一副漫画版三国杀，后来被滑稽地叫做小学生卡牌，但买时我并不会玩，只是喜新厌旧罢了。小 X 就是教那个我打三国杀的人，从武将牌教起，一步一步来。小 X 不缺时间，这样的午后似乎有很多很多。那时 小 X 高三，而我还在读小学三年级，甚至连武将的名字都不认识。我渐渐入了门，但是场场皆输。我和 小 X 说，我们玩身份牌我就会赢，你做主公我坐忠臣，我造反你就输了。剩下的，我就只记得 小 X 严肃地和我反复强调，这样做是不对的，要忠诚。  
当我已经不满足于小学生卡牌时，小 X 提出去买一套新的。于是我们去了花鸟市场，另外一家店，那套叫奇门遁甲，除了基础卡牌还有几个八卦，甚是有趣。结账的时候我有点心疼，因为要 89.9 元。对于一个星期只有十元零花钱的我来说，自然是笔不小的数目了。小 X 不在意，说自己打一次王者，开个直播就能净赚一万。听到这话，我突然觉得 小 X 好伟大。  
后来见到 小 X 的机会就很少很少了，彼此都是死宅，偶尔在爷爷奶奶家吃顿饭，不外如是。  
旁人谈起 小 X 的过往，皆是说不务正业沉迷网络一类。我每每不敢说话。  
现在奇门遁甲被完完整整地放在我的书架里，作为书架里唯一一样不是书的玩意。它上面有一层灰。

我想：人啊，降临时皆为白纸，至于是否沾染墨色，全看作画之人了。

最后呢，小 X 找到了一份得体的工作。祝工作顺利，然后有个新的开始吧。
