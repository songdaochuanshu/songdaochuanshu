## 📝 功能详情

### 🔍 搜索功能
- **目标**：支持按标题、描述、正文内容搜索文章
- **方案**：前端本地搜索，基于 manifest.json 的元数据 + 可选的全文索引
- **涉及文件**：新增搜索组件，`index.vue` 添加搜索入口

### 📑 文章目录 (TOC)
- **目标**：详情页自动生成 h2/h3 锚点导航，长文章快速跳转
- **方案**：解析渲染后的 HTML 提取标题，生成侧边或顶部目录
- **涉及文件**：`pages/posts/[...slug].vue`，新增 TOC 组件

### ⏱ 阅读时间估算
- **目标**：文章卡片和详情页显示预估阅读时间（如"约 5 分钟"）
- **方案**：基于正文字符数 / 平均阅读速度（~400字/分钟）
- **涉及文件**：`pages/posts/[...slug].vue`，`pages/index.vue`，manifest 数据处理

### 🏷 标签系统
- **目标**：展示文章标签，支持按标签筛选
- **方案**：manifest 中已有 tags 字段，需在卡片展示 + 分类栏增加标签筛选
- **涉及文件**：`pages/index.vue`，`pages/posts/[...slug].vue`

### ⬆ 回到顶部按钮
- **目标**：滚动超过一屏后显示，点击平滑回顶
- **方案**：监听 scroll 事件，fixed 定位按钮 + 过渡动画
- **涉及文件**：新增组件或直接写入布局

### 📊 阅读进度条
- **目标**：详情页顶部细条，实时显示阅读进度百分比
- **方案**：监听滚动计算 `(scrollTop) / (scrollHeight - clientHeight)`，fixed 顶部进度条
- **涉及文件**：`pages/posts/[...slug].vue`

### 🌙 暗色模式
- **目标**：支持跟随系统偏好 + 手动切换
- **方案**：Tailwind dark mode + localStorage 持久化偏好
- **涉及文件**：全局布局，所有页面的颜色 class 适配

### 🎭 页面切换动画
- **目标**：路由切换时有淡入淡出过渡
- **方案**：Nuxt 的 `<NuxtPage>` 配合 CSS transition 或 `pageTransition`
- **涉及文件**：`nuxt.config.ts`，全局 CSS

### 📦 归档页
- **目标**：按年月时间线展示所有文章
- **方案**：新增 `/archive` 路由，按日期分组渲染
- **涉及文件**：新增 `pages/archive.vue`

### 🔗 相关文章推荐
- **目标**：详情页底部推荐 2-3 篇同分类/标签的文章
- **方案**：基于 category + tags 计算相似度，取 top N
- **涉及文件**：`pages/posts/[...slug].vue`

### 📡 RSS 订阅
- **目标**：生成标准 RSS/Atom feed
- **方案**：Nuxt 服务端生成 XML，或构建时生成静态 feed 文件
- **涉及文件**：新增 server route 或构建脚本

### 🚫 自定义 404 页
- **目标**：访问不存在的路径时展示友好页面
- **方案**：Nuxt `error.vue` 或 `pages/[...slug].vue` 兜底
- **涉及文件**：新增 `error.vue`

### 🖼 文章封面图
- **目标**：卡片展示文章缩略图，提升视觉丰富度
- **方案**：manifest 增加 cover 字段，卡片布局调整为图文混排
- **涉及文件**：`pages/index.vue`，manifest 数据结构
