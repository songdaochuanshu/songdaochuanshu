上班摸鱼玩塔防，竟然是为了学架构？「GitHub 热点速览」
==============================

如果你的生产环境用了 NextJS——基于 React 的流行开源 Web 开发框架，那上周末可能过得并不轻松。**受 React 高危漏洞的影响，攻击者可以通过 HTTP 请求攻击 NextJS 项目，远程执行任意指令**。如果你不确定项目是否受影响，可以使用官方提供的工具，一条命令 `npx fix-react2shell-next` 即可检查和修复该安全漏洞。

> GitHub 地址→[github.com/vercel-labs/fix-react2shell-next](https://github.com/vercel-labs/fix-react2shell-next)

接下来，让我们看看上周 GitHub 又有哪些好玩的开源项目。近期，微软开源的支持 90 分钟高质量长语音合成框架，连续多天登顶热榜。让你在“摸鱼”时学习云架构知识的塔防游戏 server-survival，除了界面有些简陋但真能学到东西。如果你在寻找 Vim 的终端编辑器替代品，可以试试最近很火的新项目 Fresh，没点东西也不可能一周 Star 破千。

最后是无需客户端、浏览器访问就能用的远程桌面工具 CrossDesk，以及完全免费、开源的 Android 白噪音应用 XMSLEEP，帮你放松、助你入眠。

1\. 热门开源项目
----------

### 1.1 扮演云架构师的塔防游戏：server-survival

**主语言：JavaScript**，**Star：2.3k**，**周增长：2k**

这是一个基于 Web 的塔防模拟游戏，旨在通过玩游戏的方式学习云架构知识。玩家在游戏中扮演云架构师，利用有限的预算搭建云基础设施。面对持续增长的访问流量和 DDoS 攻击，需要合理规划和部署资源，尽可能延长云服务稳定运行时间（生存更长时间）。

> GitHub 地址→[github.com/pshenok/server-survival](https://github.com/pshenok/server-survival)

### 1.2 秒开大文件的终端编辑器：Fresh

**主语言：Rust**，**Star：1.4k**，**周增长：1.3k**

这是一款支持鼠标操作的终端文本编辑器，能够流畅编辑 GB 级大文件。它开箱即用、上手简单，内置 LSP 客户端，提供自动补全、跳转到定义等 IDE 功能，以及 TypeScript 插件系统。

> GitHub 地址→[github.com/sinelaw/fresh](https://github.com/sinelaw/fresh)

### 1.3 开箱即用的文件传输工具：alt-sendme

**主语言：Rust、TypeScript**，**Star：4k**

这是一款采用 Rust 和 Tauri 构建的开源 P2P 文件传输工具，无需注册和服务器，即可实现端到端的文件传输。用户只需将文件拖入应用，将随机生成的传输凭证发给接收方，对方输入后即可轻松接收文件。

> GitHub 地址→[github.com/tonyantony300/alt-sendme](https://github.com/tonyantony300/alt-sendme)

### 1.4 一句话生成小红书的创作工具：RedInk

**主语言：Python**，**Star：3.2k**，**周增长：1k**

该项目是基于 AI 的一站式小红书图文生成工具，只需输入一句话描述主题，并上传主页截图作为风格参考，即可自动生成包含封面图和内容的完整图文笔记，支持重新生成单页内容、大纲预览和编辑等功能。

> GitHub 地址→[github.com/HisMax/RedInk](https://github.com/HisMax/RedInk)

### 1.5 长时长多角色语音合成框架：VibeVoice

**主语言：Python**，**Star：14k**，**周增长：4k**

该项目是微软开源的文本转语音框架，旨在解决传统 TTS 系统在生成长篇幅、多角色对话时（博客、有声书）的痛点。它能够基于文本，一次性合成长达 90 分钟、4 位不同角色对话的高质量长音频，支持中文、英语等 9 种语言。

> GitHub 地址→[github.com/microsoft/VibeVoice](https://github.com/microsoft/VibeVoice)

2\. HelloGitHub 热评

在此章节中，我们将为大家介绍本周 HelloGitHub 网站上的热门开源项目，希望你能从中收获开源神器和编程知识，更欢迎与我们分享你在使用这些**开源项目的体验和评价**，用最真实反馈为开源项目的作者提供动力。

### 2.1 支持 Web 端的远程桌面工具：CrossDesk

**主语言：C++**

这是一款开源、轻量级的跨平台远程桌面工具，支持不同设备间（Windows、Linux 和 macOS）的远程桌面控制和音视频传输，特点是可直接通过浏览器控制远程设备，无需额外安装任何应用。

> 项目详情→[hellogithub.com/repository/kunkundi/crossdesk](https://hellogithub.com/repository/kunkundi/crossdesk)

### 2.2 开源的 Android 白噪音应用：XMSLEEP

**主语言：Kotlin**

这是一个专注于白噪音播放的 Android 应用，提供雨声、篝火、雷声、猫咪呼噜、鸟鸣、夜虫等多种自然声音，帮助你放松、冥想和入睡。

> 项目详情→[hellogithub.com/repository/Tosencen/XMSLEEP](https://hellogithub.com/repository/Tosencen/XMSLEEP)
