搜索文件排除项设置
---------

有些文件你只需要浏览，无需编辑比如自动生成的项目文件、压缩后的脚本或构建产物。这些文件不仅让搜索结果变得杂乱，还会拖慢查找速度。

现在可以将它们从`在文件中查找`和`快速查找`的搜索范围中排除，请转到 `工具`\=>`选项`\=>`环境`\=>`搜索`，然后从搜索结果部分查找新的`排除文件`。

![image](https://img2024.cnblogs.com/blog/1336199/202509/1336199-20250912074817332-1121035279.png)

编辑器底部菜单支持自定义
------------

现在我们可以直接在编辑器底部点击鼠标右键来自定义编辑器底部菜单，当然你也可以通过`工具`\=>`文本编辑器`\=>`常规`\=>`显示`\=>`在编辑器中显示编辑上下文`中管理这些设置。

GitHub Copilot 新增功能
-------------------

### 自适应粘贴

自适应粘贴功能现已上线，它能自动调整你粘贴的代码，使其更好地融入现有代码的上下文，从而帮你节省时间、减少手动修改。该功能还适用于多种场景，例如：修复小错误、统一代码风格、格式化代码等任务。

### 上下文菜单中的 Copilot 操作

现在你可以在 Visual Studio 的上下文菜单中直接与 Copilot 交互！这项新功能让你只需单击一下，即可快速获取 Copilot 对常见任务的帮助，无需手动输入任何提示。

![image](https://img2024.cnblogs.com/blog/1336199/202509/1336199-20250912074927510-2106143199.png)

动作

选中代码时

未选中代码时

解释（Explain）

解释你选中的代码段，帮助你理解其功能和逻辑

解释光标附近或当前行的代码，提供局部代码片段的理解

优化选择（Optimize Selection）

分析并优化你选中的代码段，可能包括提高性能、减少冗余、改进可读性等

不适用（N/A）

生成注释（Generate Comments）

为选中的代码自动生成详细的注释，帮助其他开发者更好地理解这段代码的目的和工作原理

为光标附近的代码生成注释，通常覆盖当前函数或代码块

生成测试（Generate Tests）

根据选中的代码生成相应的单元测试用例，帮助确保代码的正确性和健壮性

为光标附近的代码生成测试用例，通常针对当前函数或类

添加到聊天（Add to Chat）

将选中的代码片段发送到 Copilot Chat 中作为参考，以便进一步讨论、修改或获取更详细的建议

将整个文件的内容发送到 Copilot Chat 中作为参考，适用于需要对整个文件进行讨论或审查的情况

### Copilot 中支持直接渲染 Mermaid 图表

现在你可以在 Visual Studio 的 Markdown 编辑器和 Copilot Chat 中直接渲染 Mermaid 图表了！

![image](https://img2024.cnblogs.com/blog/1336199/202509/1336199-20250912075012093-1912068513.png)

### 新增 Copilot 网页链接上下文功能

现在你可以将网页链接粘贴到 Copilot 聊天中，即可让 Copilot 获取页面内容，从而为你提供更准确的答案。当 Copilot 自身的知识库没有涵盖某个主题，或者你想参考某个具体网页时，这个功能就能帮你补全信息。

![image](https://img2024.cnblogs.com/blog/1336199/202509/1336199-20250912075023622-1211803885.png)

![image](https://img2024.cnblogs.com/blog/1336199/202509/1336199-20250912075030351-1092797984.png)

### 新增性能分析助手 Copilot Agent

一个由 AI 驱动的性能分析助手，能自动分析程序性能、提出优化建议，并帮助你快速迭代改进。

![image](https://img2024.cnblogs.com/blog/1336199/202509/1336199-20250912075530995-1424455477.png)

支持文本可视化工具中搜索
------------

现在你可以在`文本可视化工具`中使用全新的`快速查找`功能，搜索字符串内容！只需在打开文本查看器时按下 `Ctrl + F`，即可调出搜索框。

![image](https://img2024.cnblogs.com/blog/1336199/202509/1336199-20250912075543960-1022564614.png)

.NET 10 和 C# 14 已内置支持
---------------------

最新版的 Visual Studio 已原生支持 .NET 10 和 C# 14！这意味着你可以立即在项目中使用最新的语言功能、性能优化和框架改进，无需额外安装或复杂配置。
