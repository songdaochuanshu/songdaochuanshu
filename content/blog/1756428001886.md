前言
--

在数字化办公日益普及的今天，高效、灵活、可定制的在线文档编辑工具已成为现代工作流中不可或缺的一部分。今天大姚给大家分享一个 .NET 开源、功能强大的在线文档编辑器，并且支持信创：DCWriterLite。

项目介绍
----

DCWriterLite 是一个基于 .NET 开源（Apache-2.0 license）、功能强大的在线文档编辑器，与 TinyMCE 和 CKEditor 等基于浏览器的 HTML DOM 开发的其他软件不同，DCWriterLite 构建自己的 DOM 结构，使用 XML 存储文档，并使用 Canvas 进行文档渲染，使用 SVG 进行文档打印。这使得 DCWriterLite 能够克服 HTML DOM 的局限性并实现高度先进的文档编辑功能，例如实时分页、可靠的重做/撤消功能和强大的表单字段。

应用场景
----

*   党政机关电子公文系统
*   医疗健康领域电子病历系统
*   金融行业合规文档处理
*   教育行业在线教学与作业系统
*   开发者二次开发与系统集成
*   等等等......

项目特点
----

*   使用CANVAS来绘制文档界面，使用SVG来进行打印。
*   深度魔改 Blazor Webassembly9.0 开发的。
*   展示了一种编程技巧，将WinForm.NET程序移植到WASM开发。
*   表单录入和自由文本录入完美融合，适用于企业级开发。
*   支持信创，能运行在Windows、Linux、MacOS、iOS、安卓、统信、麒麟、方德等操作系统。
*   纯前端组件，完全自定义的文档DOM结构，而不是类似TinyMCE/CKEditor等基于HTML DOM 开发的。

运行环境
----

*   支持 Chrome、Firefox 等浏览器。
*   支持 Windows、Linux、MacOS、iOS、安卓、统信、麒麟、方德等操作系统。
*   纯前端组件，不依赖服务器端，不依赖任何第三方组件。
