Windos 安装 Claude Code

假如您的电脑已经安装了 Claude Code 那么这一步可以忽略，直接跳过看下面步骤：

### Claude Code 介绍

Claude Code 是一个代理编码工具，可以读取你的代码库、编辑文件、运行命令，并与你的开发工具集成。可在终端、IDE、桌面应用和浏览器中使用。同时它也是一个由 AI 驱动的编码助手，可帮助你构建功能、修复错误和自动化开发任务。它理解你的整个代码库，可以跨多个文件和工具工作以完成任务。

### Node.js 安装

*   Node.js 版本 >= 18

> Claude Code 这款命令行工具本身是使用 JavaScript/TypeScript 语言开发的。Node.js 作为一个 JavaScript 运行时环境，为这类应用提供了运行所需的基础。

*   Node.js 下载：https://nodejs.cn/en/download
