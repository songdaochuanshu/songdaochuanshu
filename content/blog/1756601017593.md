云闪付联合HarmonyOS SDK打造更便捷安全的支付体验
==============================

云闪付是由各商业银行、产业各方与中国银联共建共享的移动支付产品，通过聚合银行业资源与银联广阔网络，覆盖更广阔的服务场景，如商超、食堂、菜场、公交等，搭建更开放全面的服务平台，为用户提供安全、便捷的支付服务。

用户通过云闪付App可绑定和管理各类银行账户，并使用各家银行的移动支付服务，包括二维码支付、无卡支付、转账、手机闪付、乘车码等移动支付服务，畅享各类优惠权益。

云闪付接入了华为的智感支付，智感支付是华为基于AI技术和智能传感器推出的创新支付功能，旨在简化支付流程并提升支付效率。用户只需将手机靠近支持NFC或扫码的支付终端（POS机、扫码枪等），系统即可自动识别并调出云闪付App付款码界面，无需手动打开应用。

云闪付扫码绑卡业务接入了HarmonyOS SDK [场景化视觉服务](https://developer.huawei.com/consumer/cn/sdk/vision-kit/?ha_source=hms1 "场景化视觉服务")（Vision Kit）[卡证识别](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/vision-cardrecognition?ha_source=hms1 "卡证识别")组件，可以提供身份证、行驶证、驾驶证等结构化的服务，包含自动分类功能，可自动判断卡证所属的类型，并且返回结构化信息和卡证图片信息。用户在绑卡时只需要将相机对准卡证就可以自动识别并录入卡号，大幅简化绑卡流程，提升了用户体验与绑卡成功率。

云闪付扫一扫业务接入了HarmonyOS SDK [统一扫码服务](https://developer.huawei.com/consumer/cn/sdk/scan-kit?ha_source=hms1 "统一扫码服务")（Scan Kit），作为软硬协同的系统级扫码服务，Scan Kit应用了多项计算机视觉和AI技术，不仅实现了远距离自动扫码，同时还对各种复杂扫码场景（如暗光、污损、模糊、小角度、曲面码等）做了识别优化。通过接入Scan Kit，能够快速构建自定义扫码界面，并支持多码识别、自动和手动调焦切换，提供了高效丝滑的扫码体验。

云闪付联合HarmonyOS SDK [媒体文件管理服务](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/medialibrary-kit?ha_source=hms1 "媒体文件管理服务")（Media Library Kit），通过嵌入[PhotoPicker组件](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/component-guidelines-photoviewpicker?ha_source=hms1 "PhotoPicker组件")，无需申请读取图片权限，即可实现图片和视频的访问和读取。支持选择特定类型的图片，如身份证、银行卡、二维码等；还可智能分析应用传入信息，推荐相关图片。比如扫码时，云闪付可以选择二维码的图片识别，减少用户找二维码的时间。

未来，华为将持续深化HarmonyOS SDK能力，依托系统级安全架构、端侧AI框架及分布式计算优势，助力云闪付构建更安全、更智能、更流畅的全场景支付体验，为用户打造高效可靠的数字服务闭环。
