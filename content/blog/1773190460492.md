OpenClaw 快速安装 Skill
-------------------

OpenClaw 的 skill 都通过 ClawHub 来管理，只需通过四条基础命令，便可快速上手安装你心仪的skills！

**1\. 安装 ClawHub CLI**

    npm i -g clawhub
    

安装完成后，即可在终端内使用`clawhub`命令进行快捷操作。

**2\. 搜索Skill**  
与claude中的搜索skill相似，使用`clawhub search “bocha web search”`搜索博查Web Search Skill：

    clawhub search "关键词"
    

可以通过关键词搜索想要的skill。  

**3\. 安装Skill**

使用命令`clawhub install <技能名称>`可以快速安装所需的skill，我们直接使用下面的指令安装博查 Web Search Skill：

    clawhub install bocha-web-search
    

安装完成后，重启OpenClaw会话即可生效。

如果安装失败，提示 `“Rate limit exceeded”` ，可以先登录 clawhub 以解除下载限制：

1.  使用命令 “`clawhub login`” 登录 clawhub ：  
    
2.  浏览器内打开 `Opening browser` 后的 url 链接，在浏览器内自动完成登录。  
    
3.  如果没有自动完成登录，则可以复制浏览器URL内 `#token=****` 中的 token  
      
    随后，在命令行中输入指令 `clawhub login --token <token>` 进行登录。  
    

**4\. 批量更新**

    clawhub update --all
    

一键将所有已安装的Skills更新到最新版。

Bocha Web Search + OpenClaw 最佳实践
--------------------------------

OpenClaw可以通过下面的四个基础 Skills，形成 **“感知→执行→调度→落地”的完整 AI Agent 能力闭环**，完美契合其“本地优先、自动化落地、中文场景适配” 的核心定位，也是从 “聊天机器人” 升级为 “能干活的本地助理” 的最小可行集（MVP）。

*   **bocha-web-search（博查联网搜索）**：信息入口。解决 OpenClaw 本地模型 “知识过时、中文理解差” 的核心痛点，为所有操作提供精准、实时的中文数据源（比如搜政策、查教程、追热点）；
*   **Agent Browser（网页自动化）**：交互抓手。把博查搜索的结果从 “文字” 变成 “可操作的网页动作”（比如搜到文档后自动打开、登录账号导出、截图保存），让信息落地；
*   **Shell（命令执行）**：本地底座。像龙虾的硬壳（Shell）一样，承接网页 / 搜索的结果，联动本地文件、工具、脚本（比如把搜索结果整理成 Markdown、调用飞书接口发文档），实现 “本地数据不跑路”；
*   **Cron（定时任务）**：自动化引擎。把单次操作变成 “无人值守的持续流程”（比如每天用博查搜竞品动态→Shell 整理成文件→定时推到工作群），真正解放人力。

除了龙虾的“硬壳”，让我们再看点更“硬核”的操作吧。当 OpenClaw 配齐基础能力底座后，还可以快速接入飞书、钉钉等办公工具，就能从本地检索助手升级为全流程办公 AI—— 你无需打开终端，直接在飞书对话框里和小龙虾对话，它会自动调用Bocha Web Search完成中文检索，还可以结合 Shell 完成文件读写等动作。

  

比如直接说`“帮我用博查搜索本周能源行业政策，整理成飞书文档发给@xxx”`，小龙虾会先通过博查搜索抓取精准的中文政策内容，再用 Shell 技能调用飞书接口自动生成文档、@对应成员；或是说`“每天下班前整理Openclaw产品更新动态，同步到xxx群”`，搭配 Cron 定时技能，全程无需手动操作。
