开源的 DeepSeek-R1「GitHub 热点速览」
============================

![](https://img2024.cnblogs.com/blog/759200/202502/759200-20250210233137572-1219660921.png)

春节假期回来，一睁眼全是王炸级的开源模型 DeepSeek-R1！

![](https://img2024.cnblogs.com/blog/759200/202502/759200-20250210233148254-666990254.png)

> GitHub 地址→[github.com/deepseek-ai/DeepSeek-R1](https://github.com/deepseek-ai/DeepSeek-R1)

DeepSeek-R1 开源还不到一个月，Star 数就飙升至冲破天际的 70k。虽然目前仅开源了模型权重，但同时发布的技术论文详细地介绍了 DeepSeek-R1 所采用的训练技术，如模型蒸馏、奖励建模等关键技术，受到了 AI 领域大牛和开源社区的一致好评。**DeepSeek-R1 选择开源，开源也选择了它**。这不仅“点亮”了自身，还为众多支持 DeepSeek-R1 的开源项目带去了更多关注。

![](https://img2024.cnblogs.com/blog/759200/202502/759200-20250210233155635-1115972551.png)

比如，本地运行各种 LLM 的工具 Ollama、一站式的自托管 AI 平台 Open-WebUI、新手友好的 LLM 微调工具库 Unsloth。除此之外，本周上榜的还有开箱即用的状态页面 Kener 和 跨平台的 PS4 模拟器 shadPS4。最后是一款轻量级的 AI 代理开发框架 agent-service-toolkit，可快速在本地构建自定义的 RAG 服务。

1\. 热门开源项目
----------

### 1.1 轻量级的 AI 代理开发框架：agent-service-toolkit

![](https://img2024.cnblogs.com/blog/759200/202502/759200-20250210233205356-1392621465.png)

**主语言：Python**，**Star：2.1k**

该项目能够帮助开发者用 Python 快速搭建和运行基于 LangGraph 框架的 AI 代理服务。它结合 FastAPI、Streamlit 和 Pydantic 等技术栈，提供了用户界面、自定义 Agent、流式传输等功能，并集成了内容审核（LlamaGuard）和用户反馈机制（LangSmith），极大地简化了 AI Agent 应用的开发和优化过程。

> GitHub 地址→[github.com/JoshuaC215/agent-service-toolkit](https://github.com/JoshuaC215/agent-service-toolkit)

### 1.2 开箱即用的状态页面：Kener

![](https://img2024.cnblogs.com/blog/759200/202502/759200-20250210233213511-1030118811.png)

**主语言：JavaScript**，**Star：3k**

这一个基于 SvelteKit 和 Node.js 构建的状态页面。它界面精美、开箱即用，提供了事件管理、国际化、自动调整时区和自定义域名等功能，并支持 Docker 一键部署。

> GitHub 地址→[github.com/rajnandan1/kener](https://github.com/rajnandan1/kener)

### 1.3 跨平台的 PS4 模拟器：shadPS4

![](https://img2024.cnblogs.com/blog/759200/202502/759200-20250210233221087-341073258.png)

**主语言：C++**，**Star：17k**

这是一个用 C++ 编写的 PlayStation 4（PS4）模拟器，支持在 Windows、Linux 和 macOS 系统上玩 PS4 游戏。它仍处于早期开发阶段，但社区活跃，已经能够成功运行《血源诅咒》和《黑暗之魂II》等游戏。

> GitHub 地址→[github.com/shadps4-emu/shadPS4](https://github.com/shadps4-emu/shadPS4)

### 1.4 一站式的自托管 AI 平台：open-webui

![](https://img2024.cnblogs.com/blog/759200/202502/759200-20250210233236216-177420567.gif)

**主语言：JavaScript**，**Star：70k**

这是一个功能强大、用户友好的自托管 AI 平台，支持完全离线运行。它集成了多种大型语言模型运行器，如 Ollama 和 OpenAI 兼容的 API，支持网页搜索、本地 RAG 集成、权限管理、适配移动端、Markdown 和 LaTeX 等功能。

> GitHub 地址→[github.com/open-webui/open-webui](https://github.com/open-webui/open-webui)

### 1.5 新手友好的 LLM 微调工具库：unsloth

![](https://img2024.cnblogs.com/blog/759200/202502/759200-20250210233244735-2097996041.png)

**主语言：Python**，**Star：25k**

该项目是用于微调和优化大型语言模型（LLM）的 Python 工具库。它通过动态量化和显存优化技术，提高了模型微调速度，同时将显存占用降低 70%-80%，并支持多种硬件配置、LLM、超长上下文任务等功能。除此之外，还提供了可直接在线体验的 Jupyter Notebook 示例，降低了大模型微调的门槛。

> GitHub 地址→[github.com/unslothai/unsloth](https://github.com/unslothai/unsloth)

### 2.1 沉浸式的电子书阅读器：Readest

![](https://img2024.cnblogs.com/blog/759200/202502/759200-20250210233300362-455147098.png)

**主语言：TypeScript**

这是一款为热爱阅读的用户量身打造的阅读软件，将极简设计与强大功能融合，为你带来专注、沉浸的阅读体验。它基于 Next.js 和 Tauri 开发，支持跨平台运行，现已支持 macOS、Windows、Linux、Android 和 Web 平台，未来还将推出 iOS 版本，实现真正的全平台覆盖。

> 项目详情→[hellogithub.com/repository/8a5b6ade2aee461a8bd94e59200682a7](https://hellogithub.com/repository/8a5b6ade2aee461a8bd94e59200682a7)

### 2.2 集成多协议的远程连接管理工具：mRemoteNG

![](https://img2024.cnblogs.com/blog/759200/202502/759200-20250210233307837-2118850884.png)

**主语言：C#**

这是一款功能强大的远程连接管理工具，支持 RDP、VNC、SSH 等多种主流协议。它提供了标签式界面，用户可同时管理和切换多个远程连接，支持 Windows 11、10 等系统。

> 项目详情→[hellogithub.com/repository/007333a351a44c679b38b24fe825a522](https://hellogithub.com/repository/007333a351a44c679b38b24fe825a522)
