易语言 -- 开山篇章
===========

易语言简介
=====

易语言（EPL）是一门以中文作为程序代码的编程语言，其以“易”著称，创始人为吴涛。易语言早期版本的名字为 E 语言，也通常代指与之对应的集成开发环境。其最早的版本发布可追溯至 **2000 年 9 月 11 日**。创造易语言的初衷是进行用中文来编写程序的实践，方便中国人以中国人的思维编写程序，并不用再去学习西方思维。易语言的诞生极大地降低了编程的门槛和学习的难度。

自 2000 年以来，易语言已经发展到一定的规模，在功能上、用户数量上都十分可观。

须知
--

1.  易语言不能安装到 C 盘；
2.  需要关闭电脑管家等杀毒软件，可以安装火绒安全；在首页 -> 信任区 -> 添加所有盘符；
3.  受易语言架构 32 位的限制；最大只能使用电脑的 4GB 运行内存；
4.  编译的程序容易报毒，原因为易语言的编译器的缘故；
5.  网络上很多的易语言源码需要斟酌使用；建议使用官方推荐的源码；
6.  易语言的默认编码通常是 GBK 或 GB2312，在处理 UTF-8 编码的文件或数据时可能会出现乱码；
7.  易语言主要针对 Windows 平台开发，缺乏对 Linux、macOS 等系统的原生支持；
8.  编辑器 IDE 支持不太好，因为是集成式开发环境；只能使用易语言提供的编辑器进行开发；
9.  社区生态较小，通常遇见问题只能在论坛、贴吧、QQ 群中解决；不过现在 AI 对易语言也支持一些；
10.  不支持多人协作开发！无法使用 Git、SVN 这样的版本控制工具；
11.  学习资源较少：网络上的视频资源比较老，B 站等教学的参差不齐。

**总结**：每个语言的存在都是有它合理的地方；我们用合适的语言做合适的事就好；拿着 C 与 Java 比效率；用 CSS 跟易语言去比画界面；要是实在闲得没事；把驴替下来；你背着磨盘上。
