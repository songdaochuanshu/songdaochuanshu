OpenClaw 是什么？
-------------

OpenClaw 是一款本地优先、完全开源的新一代 AI 助手。它直接部署在你的 macOS、Windows 或 Linux 设备（及服务器）上，将大模型的智慧与您的操作系统深度融合。它不仅能理解上下文，更拥有长时记忆与系统级权限，能真正“动手”执行文件管理、数据抓取、应用操控等真实任务。

------------

*   登录地址：[https://q.qq.com/qqbot/openclaw/login.html](https://q.qq.com/qqbot/openclaw/login.html)

OpenClaw原生接入流程
--------------

### 一、安装OpenClaw开源社区QQBot插件

`openclaw plugins install @sliverp/qqbot@latest   `

### 二、配置绑定当前QQ机器人

`openclaw channels add --channel qqbot --token "xxx:xxxx"   `

### 三、重启本地OpenClaw服务

`openclaw gateway restart   `
