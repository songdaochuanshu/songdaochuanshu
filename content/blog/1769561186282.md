请在vscode中使用opencode
===================

使用Opencode时，通常需要处理文件，使用Vscode编辑器可以在内部同时操作opencode和查看相关文件（如果经常使用的是office文件，可以使用open插件打开）

！**如果你同时用其他编辑器写代码慎用**，如IDEA和VSCODE同时打开代码，代码编译会出现冲突

安装
==

### 安装Vscode

直接下载安装即可，官网地址为  
[Visual Studio Code - The open source AI code editor](https://code.visualstudio.com/)

### 安装插件

#### 安装Opencode插件

#### 安装open插件

该插件用于打开非代码格式的文件  

安装后，可使用右键`Open with default application`打开xlsx、docx、pptx等文件  

如何使用
====

### 打开Vscode

Opencode会以你打开的文件夹作为根目录界定工作范围

在需要工作的文件夹，右键打开Vscode

### 在Vscode中打开opencode

#### 在opencode中如何换行？

如果用的是windows的终端，使用`ctrl+enter`换行

如果用的vscode中打开，使用`alt+enter`换行

#### 如何复制粘贴

复制：左键涂抹要复制的内容，松开鼠标自动复制  
粘贴：右键粘贴

执行如下命令：

    # 先执行我
    reg.exe add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve
    
    # 再执行我
    taskkill /f /im explorer.exe & start explorer.exe
