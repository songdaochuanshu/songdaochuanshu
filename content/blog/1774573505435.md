【译】让开发流程更高效：为 Visual Studio 订阅用户解锁 Syncfusion
=============================================

　　几个月前，我和一位开发者交谈时，他说的一番话让我印象深刻：

　　“我喜欢开发应用程序，只是不喜欢一遍又一遍地重复构建相同的 UI 控件。”

　　这是很多团队面临的现实情况。您本想专注于业务逻辑、系统架构和自身的差异化优势，结果却把大量时间耗费在搭建表格、图表、文档导出、数据仪表盘和签名流程上。

　　如果您是 Visual Studio 订阅用户，有一项福利可以帮您改变这种状况，那就是 Syncfusion。

　　符合条件的订阅用户可免费使用该服务，无需额外付费。

　　下面我来为您讲讲它为何如此重要。

为什么是 Syncfusion
===============

　　Syncfusion 提供一套全面的：

　　- 1600 多个 UI 组件

　　- 适用于.NET、JavaScript、XAML、MAUI、Blazor 等平台的跨平台控件

　　- 用于 PDF、Word、Excel 及 PowerPoint 的文档处理 SDK

　　- 数据可视化与分析工具

　　- 数字签名 API

　　简而言之，它能帮助您更快地推出精良的企业级应用程序，而无需重复造轮子。

　　根据您的 Visual Studio 订阅等级，您可获得 6 至 12 个月的这些工具高级版本使用权限。

面向 Visual Studio 专业版与企业版订阅用户
============================

　　如果您使用的是 Visual Studio 专业版或企业版，以下为包含的功能内容。

Essential Studio 企业版
--------------------

　　期限：12 个月

　　这是全套内容：

　　- 跨平台提供 1600 余种 UI 控件

　　- 高性能表格、图表、日程控件及数据可视化组件

　　- 支持 Web、桌面端与移动端的跨平台开发

　　- 用于处理 PDF、Word、Excel 及 PowerPoint 文档解决方案的 SDK

　　- 无需安装 Microsoft Office 或 Adobe 相关软件

　　- 权益期结束后可享最高 15% 的续费折扣

　　如果您正在开发业务线应用、SaaS 产品、内部仪表盘或客户解决方案，这将大幅缩短您的交付周期。

BoldSign – 电子签 API
------------------

　　企业订阅版 | 12 个月

　　现代应用程序对数字签名工作流程的需求日益增长。BoldSign 为您提供：

　　- 12 个月订阅服务

　　- 不限量模板

　　- API 接口访问权限

　　- 审计追踪与团队管理功能

　　- 内含 50 次 API 文档额度

　　- 额外使用按每份文档 0.75 美元计费

　　- 自动续费，每月 5 美元，取消订阅则停止续费

　　非常适合将安全、自动化的签名功能直接集成到您的应用工作流程中。

Bold BI
-------

　　专业版与企业版 | 6 个月

　　无需从零搭建一切，即可获得仪表板与分析功能？

　　Bold BI 包含：

　　- 120 多种数据源连接器

　　- AI 驱动的分析能力

　　- 定时调度与导出功能

　　- 无限制用户数

　　- 符合 GDPR、SOC 2 及 HIPAA 合规要求

　　如果您曾搭建过自定义报表，并总想着“一定有更高效的方法”，那么这款产品正是您所需。

面向 Visual Studio Dev Essentials 会员
==================================

　　如果您是 Visual Studio Dev Essentials 的一员，也可享受相关权益。

Essential Studio UI 版
---------------------

　　有效期：6 个月

　　精心筛选的 Syncfusion UI 组件子集，专为跨平台应用开发打造。

　　- 1600 余种高性能组件

　　- 完整源代码

　　- 优先技术支持

　　非常适合使用 Visual Studio 社区版和 VS Code 开发专业级应用的开发者。

Syncfusion Succinctly 系列电子书
---------------------------

　　持续访问权限

　　您还可获取 20 本精选开发者电子书，内容涵盖：

　　- C#

　　- Python

　　- JavaScript

　　- Azure

　　- Git

　　- Docker

　　- 机器学习

　　- 以及更多内容

　　该权益永久有效。

可接受使用准则
=======

　　您可使用这些权益进行以下开发：

　　- 内部工具

　　- 客户解决方案

　　- Demo 与 POC

　　- 参考实现方案

　　您不得：

　　- 重新分发相关库文件

　　- 共享许可证密钥

　　- 将其用于无关的个人再分发项目

　　与往常一样，在激活您的权益时请查阅详细条款。

关于资格的简要说明
=========

　　根据具体优惠内容及有效期，上述权益适用于符合条件的 Visual Studio 企业版与专业版订阅用户，以及 Dev Essentials 成员。

　　不适用的订阅类型包括：Enterprise FTE, NFR, MVP, RD, MAICPP, MPN, ISV, Bug Bounty, MCT, Student Ambassadors, Alumni, Xbox NFR Basic, Azure DevOps Tools for Teaching, Startups, We. Comms, 和 Open Source Heroes 版。

　　点击此处了解更多权益资格详情（https://learn.microsoft.com/zh-cn/visualstudio/subscriptions/syncfusion）。

切勿错失此项权益
========

　　关于订阅权益，我此前便已说明：您已为订阅支付费用，最不可取的做法便是不去激活所包含的权益。

　　其中涵盖：

　　- 1600 余种用户界面组件

　　- 企业级文档软件开发工具包

　　- 嵌入式电子签 API

　　- 可直接投入生产的分析仪表板

　　这些总价值数千美元的开发工具，您即刻便可使用。

　　若您是 Visual Studio 订阅用户，请登录：

　　my.visualstudio.com/benefits

　　激活您的 Syncfusion 权益，开启更高效的开发工作。
