【译】GPT-5 现已在 Visual Studio 中可用
==============================

　　我们很高兴地宣布，OpenAI 的最新前沿模型 GPT-5 现已在适用于 Visual Studio 的 GitHub Copilot 中逐步推出。

　　GPT-5 在代码编写和理解方面能提供更快的响应速度和更出色的性能。此次发布为您的开发工作流带来了全新级别的能力，让从想法到实现的过程更加轻松。

　　立即尝试：点击您 IDE 中的 Copilot，打开聊天界面，然后选择 GPT-5（预览版）。

![1](https://img2024.cnblogs.com/blog/270073/202508/270073-20250821165153236-389099799.gif)

为什么 GPT-5 与您息息相关
================

　　GPT-5 是 OpenAI 迄今为止最先进的模型。它在以下方面带来了显著提升：

　　- 复杂编码任务的推理和决策能力

　　- 代码质量和可维护性，即便对于大型实现，也只需极少的提示

　　- 解释的清晰度，让您能更好地理解它所做的更改

　　从构建端到端功能，到理解不熟悉的代码库，GPT-5旨在帮助您在保持掌控的同时加快进度。

可用性
===

　　GPT-5 从今日起向所有付费 Copilot 套餐逐步推出。您可以在 Agent 模式和 Ask 模式中使用它。如果您的模型选择器中尚未出现 GPT-5，请稍后再查看，因为推出工作仍在继续。注意：随着 GPT-5 的更广泛应用，GitHub 已宣布将分阶段停用多个旧模型，详情请参见：《 Upcoming deprecation of o1, GPT-4.5, o3-mini, and GPT-4o – GitHub Changelog》。

　　针对 Copilot 企业版和商业版客户：

　　管理员必须在 Copilot 设置中启用 GPT-5。启用后，组织内的用户将能够在 Visual Studio 的模型选择器中选择 GPT-5。
