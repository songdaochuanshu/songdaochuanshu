手把手教你安装微信ClawBot （Windows系统）
============================

这次OpenClaw能进个人微信，真正有意义的，其实不只是“又多了一个新功能”。  
关键在于，用起来更方便了，可以用微信直接触发OpenClaw。  
如果你本地的OpenClaw还没跑通，模型也没配好，也无济于事。要先把基础环境弄顺，再接微信，才会生效。  
最大的功能区别在于，不是只能打开电脑去操作，而是通过微信聊天的方式，像是在给“下属”安排工作一样的交互方式。

闲话不说，直接上干货：

Step1: NodeJS 是开发语言和编译器（听不懂就当我没说），第一步安装NodeJS：

Node.js 安装包下载地址为：[https://nodejs.org/en/download](https://nodejs.org/en/download)。

**官方网站提供两个版本：**

*   **LTS（Long Term Support）：**长期支持版本，推荐用于生产环境
*   **Current：**当前最新版本，包含最新特性

建议选择 LTS 版本，因为它更稳定且有长期技术支持。

Step2：安装openclaw：

1\. 最下方搜索栏里输入“CMD”，打开命令提示符。

2\. 直接安装会报错误，如图

Windows PowerShell 的执行策略限制导致的，它默认禁止运行脚本文件（包括 npx 这类 Node.js 工具的脚本）。下面给你一套完整的解决方案：

 临时允许当前会话（推荐，用完自动恢复）

执行以下命令，为当前会话临时放开限制：

 Set-ExecutionPolicy RemoteSigned -Scope Process

如上图中最后一行所示。然后开始安装Openclaw （这些都是安装微信ClawBot必须的前置软件）

输入：npm install -g openclaw

最后输入命令 npx -y @tencent-weixin/openclaw-weixin-cli@latest install

openclaw channels login --channel openclaw-weixin

接上之后能干什么?  
场景1:想到什么就发一句  
·动作  
:在微信里直接发任务  
·核心价值  
:降低心理门槛一提高使用频率  
本质:微信取代了之前的OpenClaw专用界面，减少“切换成本”  
·注意点  
·任务能不能执行取决于OpenClaw 本地是否配置了工具和模型如果模型响应慢或工具未授权，微信里发再快也没用

场景2:顺手丢链接处理  
动作  
:看到网页、文章、任务说明一发给微信里的ClawBot  
核心价值  
:让信息处理更即时、更自然  
本质  
:把OpenClaw擅长的“接任务+调工具+输出结果”融入到日常信息流  
·注意点  
链接内容必须OpenClaw 能够访问/解析  
·大量网页或复杂文档可能需要配置额外抓取/解析工具

场景3:远程问状态  
动作  
:在微信里查询本地状态、文件、任务进度  
核心价值  
:远程控制和信息查询变得简单直观  
本质  
:微信只是“遥控器”，真正执行的是本地的OpenClaw  
注意点  
权限问题很关键(比如文件夹访问、系统任务查询)  
，安全性要注意:远程指令可能涉及敏感数据

总结一句话  
微信是提供了一个窗口，让你随时随地触发OpenClaw的能力;真正能干到什么程度，还是靠你本地的工具、权限和模型配置。
