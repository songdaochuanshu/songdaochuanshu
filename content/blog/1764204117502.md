微软发布 Godot C# 游戏开发教程：godot-csharp-essentials
============================================

对于希望踏入游戏开发领域的 C# 开发者来说，一个令人振奋的好消息是：微软在其 GitHub 上发布了一个全新的开源项目**`microsoft/godot-csharp-essentials`**:[https://github.com/microsoft/godot-csharp-essentials](https://github.com/microsoft/godot-csharp-essentials)。

这个仓库是专为 C# 开发者设计的 Godot 游戏开发课程材料，旨在帮助你利用熟悉的 C# 语言和强大的 .NET 生态，顺利进入精彩的 Godot 游戏开发世界。

###### 什么是 Godot 引擎？

Godot 是一款功能全面、开源免费的跨平台游戏引擎。它以其直观的场景驱动设计、灵活的脚本语言（支持 GDScript、C# 等）以及活跃的开发者社区而备受赞誉。无论你是想开发 2D 还是 3D 游戏，Godot 都能提供强大的支持。

###### `godot-csharp-essentials`：为 C# 开发者量身打造

虽然 Godot 的原生脚本语言 GDScript 非常简洁易学，但对于广大的 C# 开发者而言，能够使用自己精通的语言来构建游戏逻辑，无疑更具吸引力。Godot 近年来对 C# 的支持越来越完善，使得开发者可以充分利用 C# 的强类型、高性能以及丰富的类库。

`microsoft/godot-csharp-essentials` 正是为此而生。它包含了一系列课程资料，系统性地介绍了如何：

*   搭建 Godot 的 C# 开发环境。
*   理解 Godot 的核心概念并在 C# 中应用。
*   掌握使用 C# 开发游戏的核心技巧。

###### 谁适合这个项目？

*   **有经验的 C# 开发者**：如果你想将自己的编程技能扩展到游戏开发领域，这里是绝佳的起点。
*   **Godot 新手**：希望从一开始就使用 C# 作为主力开发语言的 Godot 初学者。
*   **学生与教育者**：寻找高质量、有条理的游戏开发教学资源。

###### 如何开始？

你可以直接访问该项目的 GitHub 页面： [https://github.com/microsoft/godot-csharp-essentials](https://github.com/microsoft/godot-csharp-essentials),只需 `git clone` 或下载该仓库，按照其中的说明，即可开启你的 Godot C# 游戏开发学习之旅。

该项目目前拥有超过 165 个星标，社区关注度正在持续上升。这是一个宝贵的学习资源，也是微软拥抱开源游戏开发社区的又一力证。如果你对游戏开发充满热情，不妨现在就开始探索吧！
