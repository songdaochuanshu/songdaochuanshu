#### 快速上手

如果你想本地运行，这个基于 Next.js 的前端项目安装起来也非常简单：

    # 1. 克隆项目
    git clone https://github.com/hkxiaoyao/ai-draw-nexus
    cd ai-draw-nexus
    
    # 2. 安装依赖 (推荐用 pnpm)
    pnpm install
    
    # 3. 开启生产力大门
    pnpm dev
    

不过，目前的在线版本每天有 10 次免费配额，这也很正常毕竟 API 线上的费用确实贵（上次我那个免费工具被大家两天就用欠费了，哈哈 😂）。
