项目介绍
----

FluentAvalonia 一款基于 Fluent Design 和基于 WinUI 的开源免费（MIT license）、现代化的 Avalonia UI 控件库，为开发者提供美观且功能丰富的 Avalonia 界面组件。

Avalonia 介绍
-----------

Avalonia 是一个强大的框架，使开发人员能够使用.NET创建跨平台应用程序。它使用自己的渲染引擎绘制UI控件，确保在Windows、macOS、Linux、Android、iOS和WebAssembly等不同平台上具有一致的外观和行为。这意味着开发人员可以共享他们的UI代码，并在不同的目标平台上保持统一的外观和感觉。

*   **Avalonia 开源地址：** https://github.com/AvaloniaUI/Avalonia

FluentAvalonia 使用要求
-------------------

*   FluentAvalonia 不支持 Windows 7、8/8.1。尽管在这些版本的 Windows 上运行可能仍然是可能的，但不会提供对问题的支持。

FluentAvalonia 版本

需要的 Avalonia 最低版本 (底座)

**v2.3.0** (最新)

必须使用 **Avalonia 11.2.5** 或更高版本

**v2.1.0**

必须使用 **Avalonia 11.1.0** 或更高版本

**v2.0.5**

必须使用 **Avalonia 11.0.4** 或更高版本

**更早的 v2.x 版本**

必须使用 **Avalonia 11.0** 系列版本
