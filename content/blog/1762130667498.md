### 史诗级警报：ASP.NET Core 被曝 CVSS 9.9 分漏洞，几乎所有.NET 版本无一幸免！

[https://www.cnblogs.com/netry/p/19147223/CVE-2025-55315](https://www.cnblogs.com/netry/p/19147223/CVE-2025-55315)

2025 年 10 月的微软补丁星期二更新中，ASP.NET Core 漏洞 CVE-2025-55315 引起关注，其 CVSS 评分为 9.9，预示重大风险。这个漏洞涉及 HTTP 请求走私，允许攻击者绕过安全检查。受影响的.NET 版本包括.NET 10、9、8 及老版本.NET Core 2.x。微软明确指出，没有缓解措施，唯一修复方案是升级到最新 SDK 版本或更新 NuGet 包引用。漏洞危害严重，开发者需高度重视。此文章提供了详细的漏洞解析和修复指引。官方链接也提供了更多信息。

### .NET 10 Release Candidate 2(RC2)发布

[https://www.cnblogs.com/shanyou/p/19142462](https://www.cnblogs.com/shanyou/p/19142462)

.NET 团队发布了 .NET 10 RC2，此版本为长期支持版本，提供三年支持。RC2 是正式版前的最终候选，已具备生产环境可用性。其主要更新包括 JIT 编译器优化、支持 AVX10.2 硬件加速、JSON 序列化增强和安全性改进。ASP.NET Core 与 Blazor 也有重大提升，如组件状态持久化和 API 开发的强化。跨平台支持方面，MAUI 和 WPF 增加了新功能。开发工具链标准化，支持容器化，提升了效率。此版本适合生产环境部署，建议关注官方动态以获取最新信息。

### ASP.NET Core Blazor 简介和快速入门一(基础篇)

[https://www.cnblogs.com/shenchuanchao/p/19142586](https://www.cnblogs.com/shenchuanchao/p/19142586)

Blazor 是 Microsoft 开发的开源 Web UI 框架，基于.NET，可以使用 C#和 HTML 创建单页应用程序。Blazor 有两种托管模型：Blazor Server 和 Blazor WebAssembly。Blazor Server 在服务器端运行，依赖 SignalR 进行实时交互，具有较小的下载量和快速加载速度，但不支持离线工作。Blazor WebAssembly 在客户端直接运行，能够充分利用客户端资源，支持独立部署并能在无服务器环境中运行。Blazor 适合各种现代浏览器，提供开发人员以高效开发工具和强大功能。

今日人物
----

**约翰·麦卡锡**(英语：John McCarthy，1927 年 9 月 4 日—2011 年 10 月 24 日[\[2\]](https://zh.wikipedia.org/wiki/%E7%BA%A6%E7%BF%B0%C2%B7%E9%BA%A6%E5%8D%A1%E9%94%A1#cite_note-2)[\[3\]](https://zh.wikipedia.org/wiki/%E7%BA%A6%E7%BF%B0%C2%B7%E9%BA%A6%E5%8D%A1%E9%94%A1#cite_note-3))，生于[美国](https://zh.wikipedia.org/wiki/%E7%BE%8E%E5%9B%BD)[马萨诸塞州](https://zh.wikipedia.org/wiki/%E9%A9%AC%E8%90%A8%E8%AF%B8%E5%A1%9E%E5%B7%9E)[波士顿](https://zh.wikipedia.org/wiki/%E6%B3%A2%E5%A3%AB%E9%A1%BF)，[计算机科学家](https://zh.wikipedia.org/wiki/%E8%AE%A1%E7%AE%97%E6%9C%BA%E7%A7%91%E5%AD%A6%E5%AE%B6)。他因在[人工智能](https://zh.wikipedia.org/wiki/%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD)领域的贡献而在 1971 年获得[图灵奖](https://zh.wikipedia.org/wiki/%E5%9B%BE%E7%81%B5%E5%A5%96)。实际上，正是他在 1956 年的[达特矛斯会议](https://zh.wikipedia.org/wiki/%E8%BE%BE%E7%89%B9%E7%9F%9B%E6%96%AF%E4%BC%9A%E8%AE%AE)上提出了“人工智能”这个概念。

1927 年 9 月 4 日，约翰·麦卡锡出生于美国[马萨诸塞州](https://zh.wikipedia.org/wiki/%E9%BA%BB%E8%96%A9%E8%AB%B8%E5%A1%9E%E5%B7%9E)[波士顿](https://zh.wikipedia.org/wiki/%E6%B3%A2%E5%A3%AB%E9%A0%93)。他的父亲 John Patrick 是[爱尔兰裔美国人](https://zh.wikipedia.org/wiki/%E6%84%9B%E7%88%BE%E8%98%AD%E8%A3%94%E7%BE%8E%E5%9C%8B%E4%BA%BA)，母亲 Ida Glatt McCarthy 是[立陶宛](https://zh.wikipedia.org/wiki/%E7%AB%8B%E9%99%B6%E5%AE%9B)的[犹太人](https://zh.wikipedia.org/wiki/%E7%8C%B6%E5%A4%AA%E4%BA%BA)。

他于 1948 年获得[加州理工学院](https://zh.wikipedia.org/wiki/%E5%8A%A0%E5%B7%9E%E7%90%86%E5%B7%A5%E5%AD%A6%E9%99%A2)数学学士学位，1951 年获得[普林斯顿大学](https://zh.wikipedia.org/wiki/%E6%99%AE%E6%9E%97%E6%96%AF%E9%A1%BF%E5%A4%A7%E5%AD%A6)数学博士学位。分别短暂地为普林斯顿大学、斯坦福大学、[达特茅斯学院](https://zh.wikipedia.org/wiki/%E8%BE%BE%E7%89%B9%E8%8C%85%E6%96%AF%E5%AD%A6%E9%99%A2)和[麻省理工学院](https://zh.wikipedia.org/wiki/%E9%BA%BB%E7%9C%81%E7%90%86%E5%B7%A5%E5%AD%A6%E9%99%A2)供职后，麦卡锡于 1962 年-2000 年底在斯坦福担任教授，退休后成为名誉教授。

麦卡锡发明了[LISP](https://zh.wikipedia.org/wiki/LISP)并于 1960 年将其设计发表在《[ACM 通讯](https://zh.wikipedia.org/wiki/ACM%E9%80%9A%E8%AE%AF)》上。他帮助推动了[麻省理工学院](https://zh.wikipedia.org/wiki/%E9%BA%BB%E7%9C%81%E7%90%86%E5%B7%A5%E5%AD%A6%E9%99%A2)的[MAC 项目](https://zh.wikipedia.org/wiki/MIT%E8%A8%88%E7%AE%97%E6%A9%9F%E7%A7%91%E5%AD%B8%E8%88%87%E4%BA%BA%E5%B7%A5%E6%99%BA%E6%85%A7%E5%AF%A6%E9%A9%97%E5%AE%A4)。然而，他在 1962 年了离开麻省理工学院，前往[斯坦福大学](https://zh.wikipedia.org/wiki/%E6%96%AF%E5%9D%A6%E7%A6%8F%E5%A4%A7%E5%AD%A6)并在那里协助建立了[斯坦福人工智能实验室](https://zh.wikipedia.org/w/index.php?title=%E6%96%AF%E5%9D%A6%E7%A6%8F%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD%E5%AE%9E%E9%AA%8C%E5%AE%A4&action=edit&redlink=1)，成为 MAC 项目多年来的一个友好的竞争对手。[\[5\]](https://zh.wikipedia.org/wiki/%E7%BA%A6%E7%BF%B0%C2%B7%E9%BA%A6%E5%8D%A1%E9%94%A1#cite_note-%E7%A7%91%E5%AD%B8%E4%BA%BA-5)
