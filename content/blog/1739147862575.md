前言

WPF介绍
-----

WPF 是一个强大的桌面应用程序框架，用于构建具有丰富用户界面的 Windows 应用。它提供了灵活的布局、数据绑定、样式和模板、动画效果等功能，让开发者可以创建出吸引人且交互性强的应用程序。

---------

帮助开发者发现功能强大、性能优越、创新前沿、简单易用的C#/.NET/.NET Core优秀项目和框架，无论你是寻找灵感、学习新技术、改进代码质量，还是想拓展自己的技术视野，都能为你提供有价值的参考和指导。关注优秀项目和框架精选能让你及时了解C#、.NET和.NET Core领域的最新动态和最佳实践，提高开发效率和质量。**坑已挖，欢迎大家踊跃提交PR推荐或自荐（让优秀的项目和框架不被埋没🤞）**。

MaterialDesignInXamlToolkit
---------------------------

MaterialDesignInXamlToolkit 是一个开源、易于使用、强大的 WPF UI 控件库，旨在帮助开发人员在 C# 和 VB.Net 中实现 Google 的 Material Design 风格的用户界面。该框架提供了一组丰富的控件、样式和效果，使开发人员能够轻松创建现代化、具有吸引力的应用程序。

*   开源地址：[https://github.com/MaterialDesignInXAML/MaterialDesignInXamlToolkit](https://github.com/MaterialDesignInXAML/MaterialDesignInXamlToolkit)
*   详细介绍：[https://mp.weixin.qq.com/s/LpG7T0tZDisSVTvQBxK6Fw](https://mp.weixin.qq.com/s?__biz=MzIxMTUzNzM5Ng==&mid=2247498671&idx=1&sn=3034c3c23ebeb60f83fea60891ba021f&scene=21#wechat_redirect)

HandyControl
------------

HandyControl是一套WPF控件库，它几乎重写了所有原生样式，同时包含80余款自定义控件。使用HandyControl你可以轻松地创建一个美观的WPF应用程序，从而大大提高开发效率。

*   开源地址：[https://github.com/handyOrg/HandyControl](https://github.com/handyOrg/HandyControl)
*   详细介绍：[https://mp.weixin.qq.com/s/6OcSyquHutFOm5IWcgLLBQ](https://mp.weixin.qq.com/s?__biz=MzIxMTUzNzM5Ng==&mid=2247493994&idx=1&sn=afaa764eb31c01fc9260673c869cc5e6&scene=21#wechat_redirect)

*   开源地址：[https://github.com/MahApps/MahApps.Metro](https://github.com/MahApps/MahApps.Metro)

Panuon.WPF.UI
-------------

Panuon.WPF.UI 是一个适用于定制个性化UI界面的组件库。它能帮助你快速完成样式和控件的UI设计，而不必深入了解WPF的 ControlTemplate 、 Storyboard 等知识。

*   开源地址：[https://github.com/Panuon/Panuon.WPF.UI](https://github.com/Panuon/Panuon.WPF.UI)
*   详细介绍：[https://mp.weixin.qq.com/s/nhiJK5ERi0y1mrf6U4ekuw](https://mp.weixin.qq.com/s?__biz=MzIxMTUzNzM5Ng==&mid=2247494604&idx=1&sn=95084320f0c47f699f3a030949268258&scene=21#wechat_redirect)

Layui-WPF
---------

LayUI-WPF是一个WPF版的Layui前端UI样式库，该控件库参考了Web版本的LayUI风格，利用该控件库可以完成现代化UI客户端程序，让你的客户端看起来更加简洁丰富又不失美感。

*   开源地址：[https://github.com/Layui-WPF-Team/Layui-WPF](https://github.com/Layui-WPF-Team/Layui-WPF)
