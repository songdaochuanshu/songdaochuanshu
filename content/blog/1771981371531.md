OpenClaw 命令速查手册
---------------

还在为记不住 OpenClaw 命令烦恼？这份**CLI 命令 + 聊天斜杠命令 + 关键路径 + 急救流程**一站式手册，帮你快速上手、高效排错，轻松驾驭全功能。

* * *

一、终端 CLI 命令
-----------

格式：`openclaw <命令>`

### 1\. 基础操作（高频必备）

命令

说明

使用场景

`openclaw --version`

查看版本号

验证安装是否成功

`openclaw --help`

查看全部命令

忘记指令时查询

`openclaw tui`

终端对话界面

新手快速聊天

`openclaw dashboard`

网页控制台

浏览器可视化管理

`openclaw restart`

重启服务

修改配置后必执行

`openclaw stop`

停止服务

临时暂停程序

`openclaw update`

更新至最新版

版本升级

### 2\. 安装与初始化

命令

说明

`openclaw onboard`

启动交互式配置向导

`openclaw onboard --install-daemon`

配置 + 安装系统后台服务（开机自启）

`openclaw configure`

重新进入配置向导

### 3\. 诊断与排错

命令

说明

使用场景

`openclaw doctor`

全面健康检查（配置、端口、依赖）

出现问题首选命令

`openclaw doctor --fix`

健康检查 + 自动修复

尝试自动解决故障

`openclaw status`

查看运行状态（网关、端口）

确认服务是否正常

`openclaw status --deep`

深度状态检测

进阶问题排查

`openclaw logs`

查看最近日志

定位错误信息

### 4\. 模型管理

命令

说明

`openclaw models list`

列出已配置模型

`openclaw models set <提供商/模型名>`

切换默认模型

`openclaw models auth login --provider <提供商>`

OAuth 登录模型提供商

`openclaw models auth paste-token --provider <提供商>`

粘贴 API Token 认证

**切换示例**

    openclaw models set bailian/qwen3-max-2026-01-23
    openclaw models set minimax/MiniMax-M2.1

命令

说明

三、关键文件路径速查
----------

`~` 在 Windows 下为 `%USERPROFILE%`

*   `~/.openclaw/openclaw.json`：主配置文件
*   `~/.openclaw/workspace/`：默认工作区
*   `~/.openclaw/skills/`：全局技能目录
*   `~/.openclaw/agents/`：Agent 数据目录
*   `~/.openclaw/agents/<ID>/USER.md`：用户偏好
*   `~/.openclaw/agents/<ID>/memory/`：记忆存储
*   `~/.openclaw/agents/<ID>/cron/jobs.json`：定时任务
*   `~/.openclaw/agents/<ID>/sessions/`：会话记录
*   `~/.openclaw/workspace/AGENTS.md`：核心提示词

四、新手标准急救流程
----------

遇到异常，按以下顺序执行：

1.  `openclaw doctor`
2.  `openclaw logs`
3.  `openclaw status`
4.  `openclaw restart`
5.  `openclaw doctor --fix`

**配置修改固定流程**  
编辑 `~/.openclaw/openclaw.json` → 保存 → 执行 `openclaw restart`

**完全重置配置**  
`openclaw configure`
