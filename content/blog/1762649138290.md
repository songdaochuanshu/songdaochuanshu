一、什么是 VibeCoding
================

Vibe Coding（氛围编程）是一种基于大型语言模型（LLM）驱动的全新编程范式。它最大的特点是完全由AI辅助开发，开发者只需用自然语言（如普通话、英语，甚至画图或举例）向AI描述你的需求或想要实现的功能，AI就会自动生成对应的代码、界面甚至逻辑，开发者关注点从“怎么写代码”变为“看结果是否满足需求”。

**核心理念与流程：**

*   开发者用自然语言提出需求
*   AI（如ChatGPT、Copilot等）自动生成可运行的代码
*   开发者只需检验运行效果，对结果满意则接受，不满意则继续提需求，循环优化，直到符合预期
*   整个开发过程变成“提想法 —> 看效果 —> 补反馈 —> 看新效果”的闭环

**Vibe Coding 本质**：

*   开发者本质上负责“提需求+审核”，AI负责“写代码+修改”
*   目标是让普通人都能用自然语言开发软件，降低技术门槛
*   这种方式下，开发者不需亲自编写甚至理解底层代码，只需聚焦功能与结果

**优点：**

*   门槛极低，非程序员也能通过对话快速实现应用原型、自动化或小型项目
*   开发效率极高，免去大量繁琐的调试和语法
*   注重快速原型和需求验证

**潜在风险：**

*   代码质量和安全性高度依赖AI模型
*   由于开发者不直接过问代码细节，可能出现难以察觉的BUG或不合规范实现
*   不适合复杂底层基础设施或对稳定性/可靠性有极高要求的场景

总的来说，Vibe Coding用一句话总结：**让编程变成“聊天+效果验证”的极致高效生产方式，AI做体力活，人类负责拍板和需求。**

五、接入使用
======

**1\. Claude Code**

*   通过 `pnpm install -g @anthropic-ai/claude-code` 安装。
*   环境变量设置：
    *   `ANTHROPIC_BASE_URL=https://agentrouter.org`
    *   `ANTHROPIC_AUTH_TOKEN=sk-xxx`
    *   `ANTHROPIC_API_KEY=sk-xxx`
*   获取 Token：到 [AgentRouter 控制台](https://agentrouter.org/console/token)。
*   项目目录下运行 `claude` 启动。

**2\. Codex**

*   安装：`pnpm install -g @openai/codex`
*   环境变量名为：`AGENT_ROUTER_TOKEN`
    *   配置可写入 `~/.codex/config.toml`，如：
        
            model = "gpt-5"
            model_provider = "openai-chat-completions"
            preferred_auth_method = "apikey"
            base_url = "https://agentrouter.org/v1"
            env_key = "AGENT_ROUTER_TOKEN"
            
        
    *   `~/.codex/auth.json` 需填写申请的 KEY。
*   进入项目目录，终端执行 `codex` 启动。

**3\. Gemini CLI**

*   _当前不支持 AgentRouter 路由配置，无法接入。_
*   推荐使用 RooCode 或 Qwen Code 作为替代。

**4\. RooCode**

*   VSCode 扩展市场搜索 “RooCode” 安装。
*   获取 API Key 后，新增 OpenAI Compatible 供应商：
    *   基础 URL: `https://agentrouter.org/v1`
    *   API 密钥: 控制台申请的 Key
    *   可选模型: `gpt-5`/`glm-4.5`/`glm-4.6`/`deepseek-v3.1`
*   保存配置，即可在 VSCode 中使用。

**5\. Kilo Code**

*   VSCode 扩展市场搜索 “Kilo Code” 安装。
*   同 RooCode，通过新增供应商配置 AgentRouter：
    *   基础 URL: `https://agentrouter.org/v1`
    *   API 密钥: 控制台登记的 Key
    *   可选模型同 RooCode。
*   保存后可用。

**6\. Qwen Code**

*   安装：`pnpm install -g @qwen-code/qwen-code`
*   设置环境变量：
    
        export OPENAI_API_KEY="sk-xxx"
        export OPENAI_BASE_URL="https://agentrouter.org/v1"
        export OPENAI_MODEL="gpt-5"       # deepseek-v3.1, deepseek-v3.2 亦可
        
    
*   可将上述环境变量加入 shell 配置文件自动加载。
*   项目目录下运行 `qwen` 启动。
