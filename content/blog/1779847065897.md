vibe coding（二）Where you go：一个微型 windows 桌面覆盖工具
==============================================

自打有了 [第一次](https://www.cnblogs.com/Goblinscholar/p/20107050)的实践之后，现在有什么简单需求我都会尝试自己做一做，只要不涉及API，纯本地化的工具相对来说还是比较容易的。

1\. 总结
======

得益于这次的目标非常简单，同样简单总结一下：

1.  **用时**：不到两小时。
2.  **配置**：Claude code 接 dsV4-flash
3.  **花费**：0.59￥，换算约 1400 万 token。（功能简单+有了经验，第一次 plan 就完成了绝大部分内容。）
4.  **成果**：一个 5 MB 的微型 exe 应用，随时使用和删除，功能很简单，**告诉别人你去干什么了，或者电脑在干什么**，比起工具，我更愿意称它是我针对自己需求实现的一个小玩具。

2\. 具体内容
========

首先，**应用没有专门的前端界面，打开后自动进入托盘：**

![image.png](https://img2024.cnblogs.com/blog/3708248/202605/3708248-20260527094348328-1158266846.png)

如图，可以通过快捷键 `ctrl+shift+space` 直接启动，也可以在菜单里打开，**支持启用和关闭开机自启动。**
