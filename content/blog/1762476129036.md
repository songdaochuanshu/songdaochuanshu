职场人高效录屏与剪辑指南：OBS+QuickTime实用搭配
------------------------------

2025-11-07 07:36  [AlfredZhao](https://www.cnblogs.com/jyzhao)  阅读(12)  评论(0)    [收藏](javascript:void\(0\))  [举报](javascript:void\(0\))

工作需要录制Demo视频，其实MAC系统自带的 `QuickTime Player` 本身也是支持录制的，操作也非常简单，但是呢...录制的文件实在是太太太大了，有点让人受不了。

于是，发现这款免费的录制工具 `OBS Studio` 更专业、更灵活，且能在保证清晰度的情况下，输出文件也更小。

录制好的视频需要剪辑，去掉一些纯等待无意义的片段，再使用 `QuickTime Player` 就可以轻松实现。

1.  使用 OBS Studio 录制
2.  使用 QuickTime Player 剪辑

1\. 使用 OBS Studio 录制
--------------------

OBS Studio下载地址：

*   [https://obsproject.com/](https://obsproject.com/)

笔者这里选择macOS下载：

![](https://img2024.cnblogs.com/blog/635610/202511/635610-20251106110013559-205947538.jpg)

使用时，我的录制需求是只选择某一个窗口的部分区域，如何实现呢？

首先选择添加源，可以先选择到这个窗口，比如这里以Chrome浏览器为例：

![](https://img2024.cnblogs.com/blog/635610/202511/635610-20251106110012968-847711252.jpg)

确定后，默认是显示浏览器的窗口的整个页面，但我只希望单纯显示浏览器本身的内容，不要上面那些与演示无关的标签页之类，使用`option`+光标移动红色框，即可实现自定义拖动圈定指定范围。

![](https://img2024.cnblogs.com/blog/635610/202511/635610-20251106110016119-907175908.jpg)

这样调整上边有黑框，不够自然，可以右键选择`变换` -> `比例适配屏幕`，这样可以更好的展示：

![](https://img2024.cnblogs.com/blog/635610/202511/635610-20251106110015918-1790142212.jpg)

然后点击`开始录制`即可，操作完成，当`停止录制`后。

可以在`设置`\->`输出`\->`录像路径`找到具体录制文件所在的目录。

2\. 使用 QuickTime Player 剪辑
--------------------------

QuickTime 不支持复杂剪辑，如果你以后需要复杂剪辑，加文字、加转场等，建议用 iMovie（同样MAC自带） 或 CapCut。

但是笔者目前的工作录制场景也就使用到如下三个场景，完全可以满足日常工作需求。

**场景一：快速去掉音频**

发现OBS刚测试录制的视频默认有音频，而我实际不需要音频输出。

想移除掉音频，发现很多收费剪辑软件都是付费会员才可以。

但实际上，可以直接使用`QuickTime Player`操作，`编辑` -> `移除音频`：

![](https://img2024.cnblogs.com/blog/635610/202511/635610-20251106110012777-542961761.jpg)

操作完成后，另存的视频就没有音频输出了。

**场景二：使用“修剪”功能（Trim）**

使用 `QuickTime Player` 剪辑视频并去掉中间的片段，其实很简单。

但是这个方式只能保留一段连续的视频，不能直接删除中间片段，只能裁掉前后部分。

打开视频 → QuickTime Player。

菜单栏选择 编辑 → 修剪…（或按 Command + T）。

出现黄色修剪条：

![](https://img2024.cnblogs.com/blog/635610/202511/635610-20251106110012250-1176031528.jpg)

拖动左右两端滑块，选择你想保留的部分。

点击 修剪。

保存文件（文件 → 存储）。

**场景三：拆分 + 删除中间片段**

上面场景的修剪，只能裁掉前后部分，难道 `QuickTime Player` 不支持删除中间片段？

当然不是的，研究发现，去掉中间某段内容也是非常简单的操作。

打开视频 → QuickTime Player。

将播放头移动到想要删除片段的 开始位置。

菜单栏选择 编辑 → 拆分剪辑（或 Command + Y）。

再将播放头移动到想删除片段的 结束位置，再次 拆分剪辑。

![](https://img2024.cnblogs.com/blog/635610/202511/635610-20251106110012295-1203730662.jpg)

选中中间的剪辑片段 → 编辑 → 删除（或 Delete 键）。

小技巧：当播放时，出现那个红色的光标，是可以手工拖动调整的，同时配合 `Command + Y` 快捷键可以快速实现剪辑需求，过程中如果误操作还可以使用 `Command + Z` 和 `Shift + Command + Z` 快速撤销或重做。

当然，若刚开始用实在记不住这些快捷键，也可以在`编辑`菜单里操作。

![](https://img2024.cnblogs.com/blog/635610/202511/635610-20251106110012257-2117677557.jpg)

看到这里，你就已经掌握了职场人最常用的高效录屏和剪辑能力，以后录屏剪辑不用再去求人了。
