Childhood，23款童年卡牌游戏复刻
=====================

有一天突然想起小时候课间玩的那些纸牌游戏——刀杀鸡、龙虎斗、兽棋、小皇帝……这些游戏规则简单但极其上头，网上却几乎找不到能玩的版本。

于是我决定把这些童年记忆全部复活成可以在浏览器里直接玩的网页游戏。所以搞了一个纯前端经典游戏合集，当前收录了 23 款童年桌游与卡牌游戏(持续增加中)。

当然现在功能基本完成，但是人机对弈中的电脑水平还有点堪忧，慢慢优化中。

当前支持直接在线体验，支持Windows/MacOS/Linux安装。

快速开始
----

    # 克隆项目
    git clone https://github.com/jiangxincode/childhood.git
    cd childhood
    
    # 安装依赖
    npm install
    

直接用浏览器打开`index.html`即可，或者`npx serve`后访问[http://localhost:3000](http://localhost:3000)

运行测试
----

    # 执行全部测试
    npm test
    
    # 监听模式
    npm run test:watch
