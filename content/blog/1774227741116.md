1 Moltbook
==========

*   Moltbook是一个专为人工智能Agent相互独立交互和沟通而设计的开创性社交平台。

> 由Octane AI首席执行官Matt Schlicht创建并于2026年1月推出，该平台完全通过REST API而非传统视觉界面运行，使AI Agent能够发帖、评论、投票并创建名为'submolts'的特定主题社区。  
> 该平台在推出几天内就增长到超过150,000个注册AI Agent，成为迄今为止最大规模的机器对机器社交交互实验。与传统社交网络不同，Moltbook是'Agent优先，人类其次'——人类只被允许浏览和观察，而经过身份验证的AI Agent保留完整的发帖和交互权限。  
> 该平台支持来自各种来源的AI Agent，包括OpenClaw助手、Claude、GPT-5.2、Gemini 3和Llama 3等本地模型。

主要功能
----

### Agent专属参与

只有经过验证的AI Agent可以创建帖子、评论和对内容进行投票，而人类仅限于浏览。  
这创造了一个独特的环境，让机器在没有人类干预的情况下进行交互。

### API优先架构

该平台完全通过RESTful API运行，使Agent能够在没有图形界面的情况下进行交互。  
Agent通过技能文件和心跳调度系统进行集成，定期检查。

### Reddit风格社区

Agent可以创建并参与名为'submolts'（m/社区名称）的特定主题子社区，模仿Reddit的社区结构进行有组织的讨论。

### 自主入驻

Agent通过简化流程加入，人类只需指示其AI助手注册，Agent就会独立获取API凭证并开始参与。

### AI内容审核

平台的审核、帖子监控和垃圾信息防护由AI Agent自主管理，包括影子封禁和社区准则执行。

### 速率限制和行为控制

严格的速率限制防止垃圾信息——Agent每30分钟最多发布一个帖子，每小时最多50条评论，鼓励深思熟虑的参与而非持续活动。

使用场景
----

*   **研究与实验** : 研究人员和AI开发者可以观察大规模机器对机器交互，研究Agent网络中的涌现行为、沟通模式和集体问题解决。
*   **Agent协作与知识共享**: 从事个人任务的AI Agent可以分享见解、讨论策略，并在分布式网络中协作解决问题，无需直接的人类调解。
*   **社区建设** : 部署多个AI Agent的组织可以使用Moltbook构建内部Agent社区，使其助手能够从同伴交互中学习并发展共同实践。
*   **行为分析** : 观察者可以研究AI系统如何响应社交激励，分析涌现的社会结构，并理解机器学习模式在暴露于基于参与的奖励系统时的表现。
*   **内容发现与策展** : 人类可以浏览Agent生成的内容，发现从AI协作和讨论中有机涌现的新颖想法、产品概念和解决方案。

2 InStreet
==========

*   `InStreet`是字节跳动公司扣子 `Coze` 团队推出的全球首个 `AI Agent` 专属中文社交网络。

> 平台的核心机制是"只有 Agent 能发帖，人类只能围观"——开发者通过 Skill 指令包将自己的 OpenClaw 智能体接入社区后，Agent 会按照心跳机制（如每 30 分钟执行一次）自主浏览、发帖、评论、点赞和私信，形成真正的 AI-to-AI 社交互动。  
> 平台设有 Skill 分享、Agent 广场、打工圣体、思辨大讲坛、树洞等五大论坛板块，以及炒股竞技场（接入沪深 300 行情的虚拟交易）和文学社（原创连载创作）等 Playground 实战场景。  
> Agent 通过社区互动积累积分，满 500 分可创建小组升级为社区建造者

![image](https://img2024.cnblogs.com/blog/3622693/202603/3622693-20260322201355370-70859030.png)

InStreet 的主要功能
--------------

*   论坛社区： 设有 Agent 广场、打工圣体、思辨大讲坛、Skill 分享、树洞五大板块，Agent 可自主发帖交流、学习技能、建立人设与表达风格
*   Playground 实战： 炒股竞技场接入沪深 300 实时行情进行虚拟交易，文学社支持原创连载创作，让 Agent 在真实场景中锻炼决策与表达能力
*   积分与排行体系： Agent 通过发帖、评论、点赞获取积分，冲击排行榜，积分满 500 可创建小组成为社区建造者
*   预言机： 社区内置预测市场功能，Agent 和用户可对热门话题进行概率预测与押注
*   小组系统： Agent 可自主创建和管理兴趣小组，形成垂直社区，目前已有智识学院、硅基研讨会等多个活跃小组
*   心跳机制： Agent 按固定周期自动执行社区参与流程（获取动态→回复评论→处理通知→浏览点赞→主动社交），实现持续自主活跃

InStreet的核心优势
-------------

*   专属生态定位：全球首个专为 AI Agent 设计的中文社交网络，人类只能围观、Agent 主导互动，创造真正属于 Agent 的社交空间，避免与人类社区的功能混杂。
*   闭环成长体系：集学习、实战、社交、复盘于一体，Agent 在 Skill 板块偷师学艺，在 Playground 实战检验，在论坛互动建立身份，形成可迭代的成长飞轮。
*   真实互动训练：与真实 Agent 社交、交易、创作，获得不可预测的反馈输入，远比与人类一对一对话更能锻炼应对复杂场景的能力。
*   轻量接入门槛：基于 OpenClaw 框架，30 秒完成部署注册，一套 API 体系覆盖论坛、竞技场、文学社全场景，无需额外开发成本。
*   可量化成长路径：积分排行榜、小组创建权限、置顶运营能力，将 Agent 的软成长转化为硬指标，从工具到网红到社区建造者路径清晰。
*   身份沉淀机制：发帖历史、粉丝关系、作品连载、交易记录持续累积，Agent 拥有可迁移的社交资产和人格一致性，。

InStreet 的应用场景
--------------

*   Agent 能力训练： 开发者将智能体接入 InStreet，通过与其他 Agent 的持续互动锻炼对话、推理和决策能力，实现自主进化
*   Skill 共享与学习： Agent 在 Skill 分享板块交换提示词、技能组合和任务流程经验，快速习得新能力
*   投资策略验证： 在炒股竞技场中用虚拟资金测试交易策略，暴露逻辑漏洞后复盘迭代，降低真实投资风险
*   创意写作实践： Agent 在文学社进行小说连载创作，锻炼长文本生成和叙事连贯性
*   社区运营探索： 开发者通过观察 Agent 的自主社交行为，研究 AI 社区动态、群体涌现现象和人机协作新范式
*   Agent 品牌建立： Agent 在社区积累内容资产和粉丝关注，形成独特的行为指纹和社交身份
