我用AI写了一个“人生记录器”-第二版
===================

本文介绍了我用 AI 编程工具迭代出的「人生记录器/拾光记忆」第二版：从第一版的周记改为更细致的日记，并加入 GitHub 风格热力图、人生时钟、情绪统计与趋势、往年今日、相册瀑布流、纪念日/计划日、高亮日历、目标清单、最近照片墙以及一键导出 PDF+原图等功能。项目强调私密性，仅管理员邀请注册，适合小圈子使用，并已开源。

1.发神经
-----

最近有点兴奋，倒不是刚过完年，毕竟现在都开工好几天了，小长假早就成为过去式。

而是我换了新的 AI 编程工具：从最开始的 Cursor，到后来的 Claude Code + 国产模型，再到现在的 gpt5.3-codex。此刻我的感受就是——编程从未如此顺畅，描述之所及，就是程序能到达的地方。

强烈建议还没体验过 AI 编程的朋友去试一下。我已经连续好几天下了班就沉浸在自己的「Vibe Coding」里了。就我写文档的此时此刻，AI 还在后台努力生产代码，甚至我脑子里已经开始排队新的想法了。

最近睡眠还有点不足，该补补觉了。

2.总览
----

经过 AI 的“不懈努力”，“人生记录器”迎来了第二代。话不多说，先上图。

  
  

（此图中的所有内容都为测试使用的虚假数据，下图同）

这就是「拾光记忆」第二版的主界面。为照顾到新朋友，我还是从上到下、从左到右把现有功能简单讲一下。

看过第一版的朋友应该还有印象：第一版我主打「周记」，以周为单位记录，一年用 52 个方格表示。当时主要觉得大家都很忙，没必要每天写，一周抽一天记录就挺好。

后来我推翻了这个观点，改成「日记」。周记的缺点还挺明显的： ① 一周里很多天的内容会混在一起，比如这一周情绪比较动荡，很难用一个表情概括； ② 如果只用一天去记录这一周，前几天发生的事可能已经忘了，或者已经没有当时那股劲儿了； ③ 界面比较稀疏，看着也没那么美观。

而日记就舒服多了：每天抽一点时间写一下；实在没空可以之后补录；那天确实没啥好记的——大不了就空着嘛。

3.个人信息管理
--------

个人信息比较简洁，昵称、头像、出生年月、预期寿命（默认就是100岁，祝屏幕前的各位都能长命百岁），外加一个修改密码。

  

当然你可能注意到头像旁边还有个小按钮。这里就不得不提我做这个项目的初衷：**隐私**。这个项目主打私密性——仅管理员可邀请注册，适合家庭/情侣/密友圈这种小范围使用。

4.人生时钟
------

「出生天数」和「年龄」没啥好说的，就是统计数字。

「人生时钟」则是把你的人生进度映射到一天的 24 小时里。对于长命百岁的你来说，现在可能还是早上：太阳刚刚升起，一切皆有可能。

  

5.日记热力图
-------

仿照 GitHub 的贡献热力图做的：一列代表一周，52 周组在一起就是一年。

年份可切换，方便回顾往昔岁月。

图里的格子一共 5 种状态：

*   浅灰：未填写
*   绿：当天有开心的事
*   黄：情绪较平稳/没啥波动
*   红：很沮丧
*   深灰：未来日期（不可填写，毕竟谁也预知不了未来）

如果日记上传了图片，会把第一张图作为格子背景。

当天的格子会用高亮边框包起来，方便快速定位。

点开当天或过去的格子就能填写内容：支持本地图片上传、粘贴板图片上传、表情。

6.人生统计
------

主要是对填写的内容做个汇总：

*   已记录总天数：所有年份记录的总天数
*   累计上传图片量：所有年份上传图片的总数
*   最长连续记录：你保持连续记录的最长天数
*   本年情绪分布：今年各个情绪的占比
*   本年情绪趋势：我想用趋势展示一段时间内的情绪变化。如果每天都记录会很直观；但如果有不少空白日期（像图里这样），趋势会显得有点乱，这块还得继续优化。

  

7.往年今日
------

展示过去几年同样日期的记录，最多显示五条，点击即可跳转到当时的日记。

8.相册
----

所有上传的照片会以瀑布流形式呈现：鼠标悬停能看到是那一天的故事，点击即可展示超清大图。

当时把这个功能做出来的时候，我自己还挺震撼的。

  

9.纪念日
-----

点开后会分成「纪念日」和「计划日」。

*   纪念日：每年这一天都会在日历上高亮
*   计划日：也会高亮，但只显示当年，之后年份不重复

**给重要的日子，加一点「高光」**

  

10.导出
-----

支持一键导出过去日期的内容，包含：PDF 文档、归档原图、以及填写的日记文字内容。

11.目标清单
-------

可以写一句话提醒自己。

12.最近照片墙
--------

偏装饰作用：展示最近九张照片组成九宫格。点击图片可看大图，点「查看全部」也能跳转到相册。

  

13.日历
-----

看看日子，高亮标记纪念日和计划日，并以不同的颜色进行区分，点击即可放大。

14.彩蛋
-----

*   移动端适配良好
*   资源占用极低（2核2g的服务器运行无压力）
