局域网扫码枪/局域网二维码接收工具
=================

局域网扫码枪/局域网二维码接收工具

扫码枪在识别印刷于芯片表面的二维码时，存在识别效率较低、准确率不高的问题。

相比之下，手机摄像头性能更强，结合微信扫码功能，识别速度更快、准确度更高。

因此，考虑通过微信小程序进行扫码，并利用 HTTP 或 WebSocket 等方式将识别结果实时传输至电脑端。

该方案配置简便、操作简单，并支持后台稳定运行。

使用预览
----

输入到记事本

输入到 Excel

功能特性
----

模拟键盘输入扫码结果

多开程序自动打开上一个窗口

支持的连接方式

*   HTTP
*   WebSocket

支持分割方式

*   无
*   回车
*   TAB
*   逗号（,）
*   竖线（|）
*   短横线（-）

数据转发

*   WebSocket
*   TCP
*   UDP

> 转发服务器如收到`{"ldg_ac":"input","data":"要输入的内容"}`也会触发输入功能
