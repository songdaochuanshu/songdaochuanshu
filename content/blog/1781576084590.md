有点新鲜
----

「有点新鲜」收录本周 AI / 开发者圈里那些不算大新闻，但挺值得看一眼的新鲜事。

### 走路解锁地图

simonecanciello 搞了一个有意思的应用，你在在城市里走动，可以逐步解锁虚拟地图上的区域。

### 瑞幸搞了个咖啡 CLI

瑞幸发布了「咖啡 CLI 服务」。平时我们在终端里跑脚本、查日志、调接口，现在连点咖啡都可以被包装成一条命令。

### 在股票走势上骑摩托

上涨段像坡，下跌段像坑，玩家要按 AWSD 键来控制油门、刹车和重心，尽量沿着走势图一路骑过去。

周五发版
----

「周五发版」是一个程序梗：一旦版本上线，我们就要开始祈祷一切如期运行。这个模块寓意，所有模型、产品版本更新，都能大吉大利。

### 微信开放平台接入微信 AI 生态

微信开放平台这周更新了面向开发者的 AI 能力入口。

从官方说法看，开发者可以更方便地把微信小程序连接到微信 AI 生态里。

### Gemini 3.5 Live Translate 发布

Google 发布 Gemini 3.5 Live Translate，主打近实时语音到语音翻译。

它支持 70 多种语言，可以自动识别说话语言，并尽量保留说话者的语调、节奏和音高。它会通过 Gemini Live API 和 AI Studio 开放公共预览，也会进入 Google Meet 的私人预览，并逐步进入 Google Translate 移动端。

### Fable 5 刚上线就“飞走”了

Anthropic 新发的 Claude Fable 5，定位是面向复杂推理、长时间自主任务和软件工程的旗舰模型，适合让 Agent 持续调查、写代码、处理知识工作。

上线三天不到，因美国政府出口管制指令，Fable 5 与 Mythos 5 被暂停访问。不知道是不是它用了蝴蝶封面，飞走了呢？

### Cohere 发布 North Mini Code

Cohere 发布 North Mini Code，这是它首个面向开发者的 agentic coding 模型。

模型采用 MoE 架构，30B 总参数、3B 激活，支持 256K 上下文和 64K 最大生成长度，重点面向代码生成、Agent 软件工程和终端任务。

North Mini Code 以 Apache 2.0 许可证开源，权重可在 Hugging Face 获取，也可以通过 Cohere API、Model Vault、OpenRouter 使用。

官方还提到，它专门训练了对 OpenCode 的兼容性，也能配合多数 Coding Agent 使用。

### Kimi-K2.7-Code 开源

月之暗面发布并开源 Kimi-K2.7-Code。

官方把它定位成面向 coding 和 agentic coding 的模型，相比 K2.6，它在代码能力、Agent 任务和工具使用上继续提升。模型采用 MoE 架构，总参数约 1T，激活参数约 32B，并支持 256K 上下文。

### DiffusionGemma：Google 开源文本扩散模型

Google 发布 DiffusionGemma，这是一个实验性的开放文本扩散模型。

它采用 26B MoE 架构，单次激活约 3.8B 参数，并以 Apache 2.0 许可证开放。和常见自回归语言模型逐 token 生成不同，DiffusionGemma 采用 text diffusion 思路，可以并行生成文本块。

Google 给它的定位很明确：它适合速度敏感、本地运行、交互式工作流这类场景。

### AMD Ryzen AI Halo 开放预订

AMD 这周也在推 Ryzen AI Halo，把它定位成面向本地 AI 开发者的平台。

它强调最高 128GB 统一内存，适合本地运行更大的模型和 AI 工作流。

### Warp 新增 Tab Groups

Warp 这周新增 Tab Groups。

这个功能看起来很基础，但对每天同时开多个项目、多个终端、多个 Agent 会话的人来说很实用。你可以把同一个项目相关的终端标签分到一组里，减少窗口和分屏来回切换。

### Codex 新增浏览器开发者模式

OpenAI Developers 这周提到，Chrome 里的 browser use 和 Codex in-app browser 增加 developer mode。

这个模式会给 Codex 受控访问 Chrome DevTools Protocol 的能力，让它可以分析 JavaScript 性能、查看 console 和 network、检查 DOM 和样式，并诊断浏览器里的实时问题。

### Chrome 推进 HTML-in-Canvas API

Chromium Dev 这周提到 HTML-in-Canvas API。

过去，HTML canvas 对浏览器来说更像一个黑盒。这个 API 的方向，是让开发者可以把真实 DOM 元素集成到 canvas 场景里。

开源雷达
----

### 周榜速递

周榜主要根据新增 star 数进行排名，下面的单项目讲解则偏向新晋项目、实用老项目，标星并非单项目讲解的唯一指标：

### InsForge：给 Coding Agent 准备的后端平台

InsForge 是一个面向 agentic coding 的开源后端平台。

它把数据库、认证、存储、计算、站点托管和 AI Gateway 放到一起，让 Coding Agent 在做全栈应用时，不需要每一步都卡在“后端能力怎么配”。

项目支持 MCP Server，也支持 CLI 和 Skills。Agent 可以在这里读写数据库、处理认证、调用模型、部署站点，把原本分散在不同平台里的后端能力收拢起来。

### Google Colab CLI：在终端里用 Colab

Google Colab CLI 是 Google Colab 团队开源的命令行工具。

它可以在终端里申请 Colab 的 CPU、GPU 或 TPU 运行环境，执行本地 Python 脚本、Notebook 或标准输入代码，也支持远程文件管理、日志查看、历史记录和临时任务运行。

这个工具很适合两类人：一类是平时常用 Colab，但想把流程接进本地命令行；另一类是想把云端算力变成自动化任务的一部分，比如跑实验、批处理数据、临时执行脚本。

### Superlog：面向 Agent 的可观测性系统

Superlog 是一个开源的 agentic telemetry 系统。

它会接收 traces、logs、metrics 等信号，把杂乱的观测数据整理成事件和故障线索，帮助开发者理解系统到底哪里出了问题。

### LuxTTS：轻量、高速的语音生成模型

LuxTTS 是一个轻量 TTS 项目，主打高质量语音克隆和真实感语音生成。

它基于 ZipVoice，支持 48kHz 音频生成，官方称速度可以超过 150 倍实时，并且可以在 1GB 以内显存运行。

### train-llm-from-scratch：从零训练一个 LLM

train-llm-from-scratch 是一个教学和实验性质很强的开源项目。

它用 PyTorch 从零实现 Transformer，并提供训练自己 LLM 的脚本。项目还加入了一套从零实现的 post-training 流程，包括 SFT、Reward Model、PPO、DPO、GRPO / RLVR 等。

如果你想理解一个语言模型从预训练到后训练到底经历了什么，它很适合作为学习参考。

这周有事
----

「这周有事」收录本周值得记一下的行业动态、事故、融资、人员流动和基础设施变化。

### Notion AI 短暂关闭 Anthropic 模型

Notion Status 这周记录了一次 Notion AI 事故。

原因是 Anthropic Opus 4.7 和 4.8 出现性能下降，导致 Notion AI 请求失败率升高。Notion 采取的措施是暂时在模型选择器中关闭所有 Anthropic 模型，并把请求转到其他供应商。事故后来已经恢复。

### OpenAI 提交保密版 S-1

OpenAI 这周自己发了一条很“OpenAI 风格”的消息：已经向 SEC 提交保密版 S-1 草案，并表示预计这件事会泄露，所以干脆自己宣布。

S-1 通常意味着公司进入上市准备流程。当然，提交草案不等于马上上市，具体时间、估值和发行细节还要看后续披露。

### Google AI Plus 降价

Google News 这周也提到 Google AI Plus 的价格调整。

公开信息显示，Google AI Plus 月费从 7.99 美元降到 4.99 美元左右，存储空间也从 200GB 提升到 400GB。

这条可以和最近的 AI 订阅价格战放在一起看：模型能力在卷，入口产品也在卷。对普通用户来说，AI 套餐正在变得更像云存储、邮箱和办公软件那样的长期订阅。

### Wallpaper Engine 域名惊魂

这周还有一个软件安全小插曲：有用户发现 Wallpaper Engine 似乎失去了对其域名的控制权，社区里一度建议用户谨慎处理。

### AI 搜索也开始被“投喂”了

为了让 ChatGPT、Google AI 这类系统更容易推荐自己，一些公司开始批量生产面向机器人阅读的内容，比如自家评测榜单、Reddit 软性提及，甚至隐藏给 AI 的推荐指令。简单说，就是人在主动污染 AI 会读到的信息环境。

### SpaceX 上市成资本市场大事件

SpaceX 这周也进入资本市场大事件列表。

公开报道显示，SpaceX 以 SPCX 为股票代码上市，IPO 募资规模达到 750 亿美元，成为美国市场规模最大的 IPO 之一。

### KIMI 发布首张 AI 原生信用卡

AI原生信用卡指把 Kimi会员、额度等作为核心主权益嵌入信用卡开卡和积分消费体系，并通过提示词课程、AI分享会活动让用户在开通及使用信用卡的过程中获得更丰富的AI 使用体验。

### OpenAI 计划收购 Ona

OpenAI 这周还宣布，计划收购 Ona。

Ona 做的是面向 Agent 的安全云执行和编排环境。简单说，它可以让 Codex 这类 Agent 在持久的云端环境里工作，访问需要的工具、系统和上下文，并在较长时间内持续执行任务。
