一款让 Everything 更加如虎添翼的 .NET 开源辅助工具！
===================================

前言
--

相信很多同学都应该用过 Everything 这个实用的 Windows 文件搜索神器吧，今天大姚给大家分享一款让 Everything 更加如虎添翼的 .NET 开源辅助工具：`EverythingToolbar`。

*   [5款高效的文件搜索工具，工作效率提升利器！](https://mp.weixin.qq.com/s?__biz=MzIxMTUzNzM5Ng==&mid=2247511877&idx=2&sn=39a98aaaa7f5b960c053ae3d3144455d&scene=21#wechat_redirect)
    

工具介绍
----

EverythingToolbar 是一款基于 .NET 开源（MIT License）、免费的实用辅助工具，它可以将 Everything 添加到 Windows 任务栏，从而取代了 Windows 传统的文件搜索功能，提供比 Windows 内置搜索更优越、更快速的搜索体验。

![](https://img2024.cnblogs.com/blog/1336199/202504/1336199-20250411130803793-43149273.gif)

工具特点
----

*   **无缝集成**：与 Windows 10 和 11 的主题设置保持一致，提供流畅的用户体验。
*   **高效搜索**：借助 Everything 的强大搜索能力，实现快速、准确的文件查找。
*   **自定义性强**：支持自定义过滤器和“打开方式”命令，满足用户的个性化需求。
*   **易于使用**：简单的界面设计和直观的操作流程，使得用户可以快速上手。

功能模块
----

### 亮色与暗色模式

EverythingToolbar 无缝融入 Windows 10 和 11，并根据您的主题设置进行调整。

![](https://img2024.cnblogs.com/blog/1336199/202504/1336199-20250411130842155-897846371.png)

### 自定义过滤器

EverythingToolbar 读取在 Everything 中预先定义的自定义过滤器。要启用此功能，请在 `EverythingToolbar` 设置中检查 `Use Everything filters` 选项。

![](https://img2024.cnblogs.com/blog/1336199/202504/1336199-20250411130856011-1412606463.png)

### 快速切换搜索选项

支持快速访问搜索标志（匹配大小写、匹配整个单词、匹配路径和正则表达式）可让您精确查找文件。

![](https://img2024.cnblogs.com/blog/1336199/202504/1336199-20250411130907232-1376303486.png)

### 基于正则表达式的文件关联

![](https://img2024.cnblogs.com/blog/1336199/202504/1336199-20250411130919266-59681791.png)

### 兼容性与集成

EverythingToolbar 与 Windows 10 和 11 兼容，并支持与 ExplorerPatcher 和 StartAllBack 等工具配合使用，实现更全面的任务栏集成。

![](https://img2024.cnblogs.com/blog/1336199/202504/1336199-20250411130930724-416340513.png)
