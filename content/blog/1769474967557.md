开源一个自己的作品浏览器插件ChaTab，一键提交Prompt到多个AI应用
======================================

ChaTab 简介
=========

好看也好用的chrome浏览器首页插件，已经在谷歌浏览器商店上架4个月了，代码调整到基本稳定状态了，所以打算把它开源出来。  
每天在用各种AI工具，不同平台切换，非常烦人，所以就自己做了一个这样的工具给自己用，挺好用，所以分享给有类似痛点的用户。

开发工具
====

cursor + claude code

现在软件开发的成本极大降低，再也不是手工业时代的一针一线的编码了，只要基础架构设计好，AI基本可以完成 90%的设计和编码工作，

插件功能说明
======

1.  批量Prompt提交，一键快速批量发送提示词至 ChatGPT、DeepSeek 等多个 AI 应用，提升效率。

2.  智能背景切换 ， 随机更换 Chrome 启动页背景，让每次开启浏览器都有新鲜感。

3.应用导航，下拉的时候点击icon快速链接到应用地址。

4.  左侧历史记录，点击可以快速填充历史记录到tab里的输入框中。

使用说明
====

1、谷歌浏览器商店搜索 "chatab"，或者复制链接地址安装：

[https://chromewebstore.google.com/detail/chatab/gcbcmnekbambjebgfjnbgopmgcgcealn?hl=zh-CN&utm\_source=ext\_sidebar](https://chromewebstore.google.com/detail/chatab/gcbcmnekbambjebgfjnbgopmgcgcealn?hl=zh-CN&utm_source=ext_sidebar)

2、使用前请先在浏览器中登录 ChatGPT、DeepSeek 等相关平台。插件将为您提供一键提交、多平台切换的便捷体验。

注意：这个插件会修改浏览器首页。
