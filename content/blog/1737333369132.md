【推荐】一款开源且功能丰富、技术先进的在线项目任务管理系统
=============================

项目介绍
====

DooTask是一款基于PHP + Vue开源的、功能丰富、技术先进、安全性高的开源在线项目任务管理系统，适合各种规模的团队使用。提供各类文档协作工具、在线思维导图、在线流程图、项目管理、任务分发、即时IM，文件管理等工具；同时消息功能使用非对称加密技术让你的沟通更安全。

项目作用
====

DooTask项目通过提供丰富的协作工具、优化项目管理流程、保障沟通安全、降低部署成本、促进知识共享和传承以及增强团队凝聚力等方面的作用，为团队带来了显著的价值和好处。

项目功能
====

*   文档协作工具：支持多人在线编辑文档，提升团队协作效率。
*   在线思维导图：帮助团队理清思路，规划项目结构。
*   在线流程图：支持绘制项目流程图，直观展示项目进展。
*   项目管理：提供任务分发、进度跟踪等功能，确保项目按时完成。
*   即时IM：内置即时通讯工具，方便团队成员实时沟通。
*   文件管理：支持文件的上传、下载和共享，确保团队资料的安全和便捷访问。
*   消息加密：使用非对称加密技术保护消息内容，确保沟通安全。

安装程序
====

*   必须安装：Docker v20.10+ 和 Docker Compose v2.0+
*   支持环境：Centos/Debian/Ubuntu/macOS/Windows
*   硬件建议：2核4G以上
*   特别说明：Windows 用户请使用 git bash 或者 cmder 运行命令
