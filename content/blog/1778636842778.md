🔧 先唠唠：我为啥被这块板圈粉了？
==================

很多刚入门嵌入式的朋友跟我吐槽：“想玩网络开发，一上来就要啃 C 语言、调底层网络协议栈、折腾编译链，直接被劝退了”。其实我之前也有这感觉，直到遇到了 WIZnet-EVB-Pico2+MicroPython 这套组合，才发现嵌入式网络开发原来可以这么轻松。

先给大家划几个我觉得最戳的点：

*   ​**和树莓派 Pico2 引脚完全兼容**​：手里有 Pico 扩展板的直接插上就能用，不用重新折腾硬件，新手友好度拉满！
*   ​**自带硬接线 TCP/IP 芯片**​：有 W5500、W5100S、W6300 三个版本可选，不用自己实现协议栈，网络稳定性直接拉满，W5500 还过了 AWS 物联网认证，做项目也很稳。
*   ​**MicroPython 生态完美适配**​：不用管复杂的编译流程，几行 Python 代码就能控制外设、写网络应用，把精力聚焦在 “硬件交互 + 功能实现” 上，爽到飞起！

整理的教程笔记包括什么？
============

这份我耗时整理、亲测验证的 ​**WIZnet-EVB-Pico2 全栈 MicroPython 开发教程**​，从硬件底层到项目实战，帮你打通嵌入式网络开发的全流程！

1\. 先把你的板子 “摸透”—— 硬件基础打底
------------------------

教程开篇就帮你搞定「硬件认知」的痛点：

*   详解 **WIZnet-EVB-Pico2** 开发板核心参数、引脚定义、硬件电路（电源 / 晶振 / BOOT/LED/ 调试接口全覆盖）
*   拆解 **RP2350** 芯片架构、引脚复用规则，帮你搞懂 “为什么引脚能同时支持 GPIO/I2C/SPI”
*   整理官方参考资料清单（SDK / 固件 / 数据手册 / 示例程序），不用自己到处找文档

2\. 环境搭建零门槛 —— 告别 “Hello World” 卡半天
-----------------------------------

新手最头疼的环境问题，教程一步到位：

*   手把手教你 MicroPython 固件烧录（BOOTSEL 模式、UF2 拷贝、常见问题排坑）
*   对比不同 IDE 的使用：Thonny（入门首选）、PyCharm/VS Code（进阶开发），附插件配置清单
*   推荐在线仿真平台（Wokwi/Unicorn），不用硬件也能先跑通代码

3\. REPL 交互神器 —— 调试效率直接拉满
-------------------------

MicroPython 的灵魂就是 REPL！教程帮你吃透这个工具：

*   讲清 REPL 的工作原理、常用快捷键（中断执行、软复位、历史命令回溯）
*   教你用 USB / 串口两种方式连接开发板，快速上手调试
*   从 `help()`/`dir()` 查询用法，到直接用指令点亮 LED，实操带你感受 “即时反馈” 的快乐

4\. 外设全掌握 —— 硬件原理 + MicroPython 代码双管齐下
--------------------------------------

不管是基础外设还是进阶功能，教程都讲透：

*   基础外设：GPIO、UART 串口、I2C/SPI 通信、定时器 / PWM，附 MicroPython 类的使用方法（比如 `machine.UART`/`machine.I2C`）
*   进阶功能：PIO 可编程控制、DMA 直接内存访问、OneWire 单总线、USB 设备开发
*   每个外设都讲清「硬件原理 + 代码配置 + 实操案例」，知其然也知其所以然

5\. 以太网通信核心 —— 玩转 WIZnet 模块（重点！）
--------------------------------

专门针对 WIZnet-EVB-Pico2 的以太网模块（W5500/W5100S/W6300），从理论到实战全覆盖：

*   网络基础：OSI 七层模型、TCP/IP 协议栈，帮你搞懂网络通信底层逻辑
*   模块详解：WIZnet 以太网模块的内部结构、寄存器配置、数据读写机制
*   MicroPython 实战：网络初始化、Socket 编程、TCP/UDP 通信、DHCP 自动获取 IP、HTTP/MQTT 协议实验

6\. 进阶实战落地 —— 从文件系统到 IPv6 项目
----------------------------

教程不止基础，更有可落地的进阶内容：

*   文件系统：MicroPython 文件管理、SD 卡挂载、EEPROM 数据存储，帮你搞定数据持久化
*   IPv6 开发：从 IPv4 的局限性讲起，到基于 W6300 的 IPv6 Web 服务器、Cookie 测试框架实战，直接上手项目开发

✨ 为什么推荐这份教程？
============

*   ✅ ​**从 0 到 1 的完整路径**​：硬件 → 环境 → 外设 → 网络 → 项目，一步不落，不用自己拼资料
*   ✅ ​**WIZnet 深度适配**​：专门针对 WIZnet 以太网模块的 MicroPython 开发，解决新手找不到适配教程的痛点
*   ✅ ​**理论 + 实操双在线**​：每个模块都附实操步骤和排坑指南，新手也能跟着跑通代码
*   ✅ ​**MicroPython 友好**​：避开复杂 C 语言底层，用 Python 快速上手嵌入式开发，同时讲清硬件原理
*   ✅ ​**持续更新迭代**​：后续会补充更多项目案例、进阶技巧，关注不迷路

👥 适合谁看？
========

*   嵌入式新手 / 在校学生：想入门 MicroPython + 以太网开发，找不到系统教程的
*   创客 / DIY 爱好者：想用 Pico 做带网络功能的项目（物联网设备、Web 服务器等）
*   想转 MicroPython 开发的嵌入式工程师：需要快速上手 RP2350+WIZnet 以太网的资料
*   对 IPv6 嵌入式开发感兴趣的开发者：想了解 W6300 的 IPv6 应用场景
