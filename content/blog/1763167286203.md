Nessus 10.8.5 在 Ubuntu 22.04 下的完整配置指南（含激活与突破 16IP 扫描限制）

![Nessus 10.8.5 在 Ubuntu 22.04 下的完整配置指南（含激活与突破 16IP 扫描限制）](https://img2024.cnblogs.com/blog/3717091/202511/3717091-20251114153121716-1204407870.png) 本文详细介绍了 Nessus 10.8.5 在 Ubuntu 22.04 系统下的完整配置流程，涵盖从安装到突破限制的全步骤。首先需上传安装包并通过dpkg命令安装，启动服务后通过http://IP:8834完成 Web 端基础配置；接着重点讲解离线激活操作，包括获取 Challenge Code 与 Activation Code、上传许可证并导入插件；最后说明创建扫描任务的方法，同时提供突破基础版 16 个 IP 扫描限制的方案，解决了 Nessus 默认无插件、扫描范围受限的核心问题，适用于需要在 Ubuntu 环境下使用 Nessus 进行漏洞扫描的场景。

Nessus 安装与配置指南（Ubuntu 22.04）
----------------------------

### 准备工作

1.  操作系统：Ubuntu 22.04
2.  下载安装包：`Nessus-10.8.5-ubuntu1604_amd64.deb`

[漏扫工具Nessus-10.8.5-ubuntu1604\_amd64.deb](https://cloud.kong.college/s/bKC9)  
[Ubuntu 22.0.4](https://cloud.kong.college/s/NghQ)

### 安装 Nessus

1.  将下载的安装包上传至Ubuntu服务器
    
2.  执行安装命令：
    
        dpkg -i Nessus-10.8.5-ubuntu1604_amd64.deb
        
    
    ![安装过程截图](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=d0f31d053a6fc46df22a867e7fb36eac)
    
3.  启动Nessus服务：
    
        systemctl start nessusd
        
    
4.  访问Web控制台：  
    通过 `http://服务器IP:8834` 访问，按以下顺序配置：
    
    *   勾选 `Register offline`
    *   选择 `managed scanner`
    *   选择 `tenable.sc`
    *   设定管理员账号密码
    *   等待初始化完成
    
    ![初始化完成截图](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=dbae2a127a6607f34847baf9942efae8)
    

### 登录后注意事项

> 默认情况下，Nessus未安装任何插件，需手动激活并导入插件。

### 激活 Nessus

#### 步骤1：获取 Challenge Code

1.  访问[插件获取页面](https://plugins.nessus.org/v2/offline.php)
2.  执行命令生成Challenge Code：
    
        /opt/nessus/sbin/nessuscli fetch --challenge
        
    
    ![Challenge Code生成结果](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=3cb9f7656cbd65bae4ca246ec9ddb86a)

#### 步骤2：获取 Activation Code

1.  访问[激活码获取页面](https://www.tenable.com/products/nessus/nessus-essentials)  
    ![Activation Code获取页面](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=f64ea461f54561be3540812088ceffe9)
    
2.  在插件获取页面填写信息：
    
    *   第一个输入框：填入离线获取的`Challenge code`
    *   第二个输入框：填入在线获取的`activation code`  
        ![信息填写页面](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=46f9b25887145ef925c23088b248e903)
3.  完成后会获取到：
    
    *   激活许可证（LICENSE）
    *   插件下载地址（如[插件地址](https://plugins.nessus.org/v2/nessus.php?f=all-2.0.tar.gz&u=9d8313aab88661974dc4b681aaf90042&p=0982ec81c16fd58a073901aab99bc4f4)）  
        ![获取结果截图](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=304eac53d9fb782799183cb7d5519517)

#### 步骤3：上传并配置许可证与插件

1.  将获取到的`LICENSE`文件和插件压缩包（`all-2.0.tar.gz`）上传至服务器  
    ![文件上传截图](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=c7dd61c36c93477bad9725f10456ee60)
    
2.  离线激活Nessus：
    
        /opt/nessus/sbin/nessuscli fetch --register-offline ./nessus.license
        
    
    ![激活结果截图](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=4a7ca81e56a510698b0f90df71e7e2f9)
    
3.  导入插件：
    
        /opt/nessus/sbin/nessuscli update all-2.0.tar.gz
        
    
    ![插件导入截图](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=eff01c0ab3f2141e95427349d74cf6f8)
    
4.  重启Nessus服务：
    
        systemctl restart nessusd
        
    
    > 注意：需等待插件安装完成后再进行后续操作  
    > ![插件安装完成截图](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=0fc851fc2d4043d909acfbaf4ce701cc)
    

### 创建扫描任务

1.  登录Web控制台后，按照指引创建扫描任务  
    ![创建扫描截图](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=f1b7c4922ae9519525fa392aaaa6b115)

### 突破16个IP扫描限制

默认基础版Nessus仅支持扫描16个IP，可按以下步骤突破限制：  
![限制提示截图](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=a891c69233d9865c24e52a1e698e603f)

#### 步骤1：停止Nessus服务

    systemctl stop nessusd.service
    

#### 步骤2：创建配置文件

1.  创建`plugin_feed_info.inc`文件：
    
        touch plugin_feed_info.inc
        
    
2.  编辑文件内容（注意`PLUGIN_SET`需与最新插件版本匹配）：
    
        PLUGIN_SET = "202509301708";  # 需替换为实际的插件版本号
        PLUGIN_FEED = "ProfessionalFeed (Direct)";
        PLUGIN_FEED_TRANSPORT = "Tenable Network Security Lightning"; 
        
    
    ![配置文件内容截图](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=c2fb6cf3d999e8bcdeb3e689d45ee0fa)
    
    > 如何获取`PLUGIN_SET`值？参考截图中的版本信息  
    > ![PLUGIN_SET获取截图](http://markdown.kong.college/server/index.php?s=/api/attachment/visitFile&sign=184a617e61d6ffde3e898b8561737a19)
    

#### 步骤3：替换配置文件

    cp plugin_feed_info.inc /opt/nessus/var/nessus/plugin_feed_info.inc
    

#### 步骤4：启动Nessus服务

    systemctl start nessusd
