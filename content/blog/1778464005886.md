前言
--

Windows 10/11 虽然功能完善，但在实际使用和系统部署过程中，预装应用较多、隐私选项分散、性能设置繁琐、个性化配置重复等问题，常常会增加用户和 IT 技术人员的维护成本。对于经常重装系统、批量配置电脑，或希望获得更干净、更高效 Windows 使用体验的用户来说，一款集系统精简、优化和定制于一体的工具就显得非常实用。今天大姚给大家分享一款基于 C# 开发的 Windows 10/11 系统增强工具，精简、优化、定制一站完成：**Winhance**。

项目介绍
----

Winhance 一款基于 C# 开发且开源的 Windows 10/11 系统增强工具。它通过图形化界面整合了应用管理、系统优化、隐私设置、桌面定制、常用软件安装、配置导入导出以及自定义安装介质等能力，帮助用户更高效地完成 Windows 环境配置，实现 **精简、优化、定制一站完成**。

项目应用场景
------

*   **新机初始化**：新电脑到手后，想快速卸载预装应用、安装常用软件？Winhance 可以帮你集中完成系统初始化配置。
*   **系统重装后**：重装 Windows 后，不想反复调整任务栏、开始菜单、隐私和性能设置？Winhance 支持配置文件导入，快速恢复常用配置。
*   **日常优化维护**：系统使用一段时间后，想减少不必要的系统干扰、优化性能和使用体验？Winhance 可统一管理隐私、更新、电源、通知和个性化设置。

项目功能模块
------

*   **软件管理**：支持移除或安装 Windows 应用、旧版功能和可选功能，并可通过 WinGet 快速安装常用外部软件。
*   **系统优化**：支持隐私、性能、电源、Windows 更新、通知、声音和 UAC 等系统设置调整。
*   **界面定制**：支持深色/浅色主题、任务栏、开始菜单和资源管理器等个性化配置。
*   **高级工具**：支持创建自定义 Windows ISO，并可生成 autounattend.xml 自动应答文件。
*   **配置复用**：支持保存和导入 Winhance 配置文件，方便新系统或重装后快速恢复设置。
