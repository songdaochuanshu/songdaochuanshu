# 你好，世界！

欢迎来到我的博客！🎉

## 关于这个博客

这个博客使用以下技术栈构建：

- **Nuxt 3** - 现代 Vue.js 框架
- **Cloudflare R2** - 对象存储
- **Tailwind CSS** - 实用优先的 CSS 框架
- **TypeScript** - 类型安全

## 特性

✅ 服务端渲染 (SSR)  
✅ 响应式设计  
✅ 深色模式支持  
✅ Markdown 文章管理  
✅ 标签分类系统  

## 开始写作

只需在 R2 的 `posts/` 目录上传 Markdown 文件即可！

```markdown
---
title: "文章标题"
date: "2025-01-01"
tags: ["标签1", "标签2"]
---

# 文章内容

这里是你的 Markdown 内容...
```
