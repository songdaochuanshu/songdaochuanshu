# 松岛川树

一个记录想法、技术与生活的个人博客。

## 关于这个博客

用代码搭的小窝，写一些值得留下来的文字。技术笔记、生活碎片、偶尔的胡思乱想，都在这里安家。

## 技术栈

- **Nuxt 4** + **Vue 3** — SSR 渲染
- **Cloudflare R2** — 文章存储与分发
- **Tailwind CSS 4** — 样式
- **TypeScript** — 类型安全

## 本地运行

```bash
pnpm install
cp .env.example .env
# 编辑 .env 填入 R2 配置
pnpm dev
```
